# <a name="error-404-when-accessing-files-in-sharepoint-or-onedrive"></a>Viga 404 kui juurdepääsu SharePointi või OneDrive faile

**Viga 404** saadakse, kui kasutajad proovivad pääseda sait või faili OneDrive'i või SharePointi. See on sageli tingitud saidi või faili või rühma saada ümber nimetatud, teisaldatud või kustutatud.
Kasutajatele, kellel 404 viga Root saidikogumis pääseda ja see on kustutatud.

Viga 404 lahendamiseks saidi, mis on ümber nimetatud, teisaldatud või kustutatud: klassikaline saite, mis olemas klassikaline administreerimiskeskuses, vt [taastada kustutatud saidikogumi](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection).

Kaasaegne saidid (side, rühma ühendatud, või mujal) mis on olemas uus admin keskus eelvaade, vt [vaadata ja taastada kustutatud saite uue SharePointi administreerimiskeskuses](https://docs.microsoft.com/sharepoint/view-and-restore-deleted-sites-in-new-admin-center).

Lahendamiseks faili (või muu üksuse) mis on ümber nimetatud, teisaldatud või kustutatud OneDrive'i või SharePointi saiti ja Vaata prügikasti: saidi sisu, vt [taastada üksusi, SharePointi saidi prügikasti](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be).

 Kui ei ole võimalik leida kauba otsingu auditilogi kui logimine on lubatud, vt [otsida auditilogis Office 365 turvalisuse &amp; täitmise Center](https://support.office.com/client/search-the-audit-log-in-the-office-365-security-compliance-center-0d4d0f35-390b-4518-800e-0c7ec95e946c).
