# <a name="problems-setting-up-an-email-account-in-outlook"></a>Outlookis meilikonto häälestamise probleemid

Kui teil on Outlookis Exchange ' i konto häälestamisel probleeme, saate kasutada probleemses arvutis [Outlooki konto häälestamise tööriista](https://aka.ms/SaRA-OutlookSetupProfile) . See teostab mitmesuguseid kontrollimisi ja pakub teie konto häälestamise probleemile võimalikke lahendusi.
  
Kui teil on endiselt probleeme, lugege artiklit [Outlook ei saa häälestada uut profiili Exchange Online ' i postkasti Exchange ' i automaattuvastuse abil](https://docs.microsoft.com/exchange/troubleshoot/outlook-profiles/cannot-set-up-profile-autodiscover).
