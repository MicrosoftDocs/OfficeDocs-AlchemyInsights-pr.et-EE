# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Tõrkeotsing Outlook mobile setup ühiskasutuses postkasti

Mobiilsideseadme installitud rakendusest Outlook ei pääse ühiskasutusse antud postkasti. Kas soovite selle funktsiooni? Las Outlook Engineering meeskond teada. [Hääletus siin](https://go.microsoft.com/fwlink/?linked=862116)!
  
Saate kasutada ainult ühiskasutuses postkasti kasutades avatud mobiilseadme brauseri Office 365 sisselogimine ja seejärel minna Outlook Web. Lisateabe saamiseks vaadake [Lisa ühiskasutusse antud postkasti Outlook Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Outlook Web ei unusta juurdepääsu ühiskasutuses postkasti.
  
Lisateave: [Outlooki iOS ja Android - rakenduse teave](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
