# <a name="how-to-type-special-characters-on-a-mac"></a>Kuidas tippida erimärgid Mac

Tippige oma Mac erimärgid, avage **erimärgid** menüü klaviatuuri otsetee ⌘ **käsu** + **kontrolli** + **ruumi**.

Kiire otsetee @ jaoks on ka **alt** + **64**.
