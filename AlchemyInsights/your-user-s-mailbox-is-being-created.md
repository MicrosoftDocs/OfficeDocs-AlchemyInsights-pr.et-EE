# <a name="user-mailbox-creation-is-taking-a-long-time"></a>Kasutaja postkasti loomine võtab kaua aega

Kui Uus postkast on kasutamiseks valmis, võib kuluda kuni 24 tundi.
  
Kui kasutaja postkast on "ettevalmistamisel" üle tunni, proovige eemaldada kasutajalt Exchange ' i litsents ja lisada see uuesti või [võtke ühendust toega](https://docs.microsoft.com/microsoft-365/admin/contact-support-for-business-products?tabs=online).
