# <a name="create-a-shared-mailbox"></a>Ühispostkasti loomine

Ühiskasutusse antud postkasti võimaldab rohkem kui ühe isiku saata ja vastu võtta e-posti alias, nagu **Sales@contoso.com**, ilma teise litsentsi määramist.
  
Lisateabe saamiseks vaadake [ühiskasutusega postkasti loomine](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox) .
