# <a name="add-phone-number"></a>Telefoninumbri lisamine

Enne kui saate oma organisatsiooni kasutajad häälestada telefonikõnesid tegema ja vastu võtma, peate neile [hankima telefoninumbrid](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview). Telefonisüsteemi seadistamise lõpetamiseks oleme loonud samm-sammulise protsessi. Lugege teemat [Telefonisüsteemi häälestamine oma organisatsioonis](https://docs.microsoft.com/MicrosoftTeams/phone-number-calling-plans/port-order-overview).
