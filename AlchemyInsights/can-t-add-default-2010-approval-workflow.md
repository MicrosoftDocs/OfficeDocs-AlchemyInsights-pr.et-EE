# <a name="cant-add-default-2010-approval-workflow"></a>Ei saa lisada vaikimisi 2010 kinnitamise töövoog

Microsoft SharePointi saidikogumi, ei saa lisada (nt "kinnitamine - SharePoint 2010") ülemaailmselt korduvkasutatava töövoo loendi või teegi. Kinnitamise töövoog võib puududa. Selle probleemi lahendamiseks vaadake [ei saa globaalselt korduvkasutatava töövoo SharePointi loendisse või teeki lisada](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Lisateavet töövoo kinnitused Vaata [Kõik kasutaja lähtestamine](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Teave [Kaasaegse kinnitused voolu](https://flow.microsoft.com/blog/introducing-modern-approvals).
