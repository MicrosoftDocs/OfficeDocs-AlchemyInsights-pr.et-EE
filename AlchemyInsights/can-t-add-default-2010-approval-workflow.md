# <a name="cant-add-default-2010-approval-workflow"></a>Ei saa lisada vaikimisi 2010 kinnitamise töövoog

Microsoft SharePointi saidikogumi, ei saa lisada (nt "kinnitamine - SharePoint 2010") ülemaailmselt korduvkasutatava töövoo loendi või teegi. Kinnitamise töövoog võib puududa. Lahendada see probleem viidata [ei saa globaalselt korduvkasutatava töövoo SharePointi loendisse või teeki lisada](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).
