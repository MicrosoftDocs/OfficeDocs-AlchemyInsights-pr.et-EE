# <a name="entity-type-not-showing-in-search-results"></a>Olemi tüüp ei Näita tulemusi

Üksuse kuvamiseks kategoriseeritud Otsi või asjakohasust Otsi Dynamics 365 ühtne liides, siis tuleb esimene lisatud app moodul.

Märkus: Kui otsite manuste asjakohasust otsing, manuse üksus lisatakse rakenduse moodul on peamise kirje, mida kuvatakse tulemused.
