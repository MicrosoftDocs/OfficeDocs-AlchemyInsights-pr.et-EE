# <a name="cant-create-a-compatibility-assessment"></a>Ühilduvuse hindamist ei saa luua

Microsoft Endpoint Configuration Manager on seadme häälestanud, kuid töölaua analüüs ei saa luua ühilduvuse hindamist. Seadme konfiguratsiooni (loendus) või installitud rakenduste (laovarude) jaoks pole täielikke andmeid seatud.

See probleem parandatakse sageli automaatselt, kui seade proovib uuesti. Kui see ei lahene, veenduge, et seade saaks teenusega suhelda. Lisateavet leiate teemast [lõpp-punktid](https://docs.microsoft.com/configmgr/desktop-analytics/enable-data-sharing#endpoints) ja [puuduvad andmed](https://docs.microsoft.com/configmgr/desktop-analytics/monitor-connection-health#missing-data).
