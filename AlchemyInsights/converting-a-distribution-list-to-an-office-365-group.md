# <a name="convert-a-distribution-group-to-an-microsoft-365-group"></a>Teisendage levirühm Microsoft 365 rühma

Saate teisendada levirühmade Microsoft 365 rühmad, kui nad vastavad teatud kriteeriumidele. Rühma teisendamiseks või Teisendusprobleemide tõrkeotsinguks vaadake teemat [täiendamine leviloendid Microsoft 365 rühmad Outlookis](https://docs.microsoft.com/office365/admin/manage/upgrade-distribution-lists).
