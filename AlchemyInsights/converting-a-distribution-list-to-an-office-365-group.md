# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Levirühma teisendamine Microsoft 365 rühmaks

Levirühma saate teisendada Microsoft 365 rühmadesse, kui need vastavad teatud kriteeriumidele. Rühma teisendamiseks või konversioonide tõrkeotsinguks lugege artiklit [Outlookis leviloendite värskendamine Microsoft 365 rühmadesse](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).
