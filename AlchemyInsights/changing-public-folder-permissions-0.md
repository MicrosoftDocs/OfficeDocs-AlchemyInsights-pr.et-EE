# <a name="changing-public-folder-permissions"></a>Avaliku kausta õiguste muutmine

Avalike kaustade õigusi saab muuta kasutajate ja administraatorite Outlookis. Administraatorid määrata ka õigused: selle Exchange'i Admin keskus (EAC), järgmiselt:
  
1. O365 portaalis, kasutades vasakul navigatsioonimenüü, liikuda **Admin keskused** \> **vahetus**.

2. Liikuge **avalikku kausta**.

3. Sealt saate muuta õigusi eraldi avalike kaustadega määrate turvagruppide õigused. Avalike kaustade õiguste muutmiseks kasutajal, kasutaja peab olema omaniku õigused kausta.
