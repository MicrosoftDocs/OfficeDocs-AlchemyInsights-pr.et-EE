# <a name="change-your-language-date-and-time-format-or-time-zone"></a>Keel, kuupäeva ja kellaaja vorming või ajavööndi muutmine

1. **Valige lehe ülaosas** > **kõik Outlooki sätete vaatamine**.
2. Valige **Üldine** > **keel ja aeg**.
3. Valige soovitud keel.
4. Valige **Salvesta**.

Lisateabe saamiseks vaadake [muuta ajavööndi- ja keelesätted Outlook Web](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
