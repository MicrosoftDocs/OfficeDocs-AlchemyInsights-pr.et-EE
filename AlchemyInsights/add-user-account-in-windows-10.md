# <a name="add-a-user-account-in-windows-10"></a>Kasutajakonto lisamine Windows 10-sse

Windows 10 konto lisamiseks toimige järgmiselt.

1. Avage > **sätete** >  **** > **kontode**käivitamine**pere & teised inimesed** (või **teised inimesed**, kui kasutate Windows 10 Enterprise) > **lisada keegi teine sellesse arvutisse**.
2. Järgige juhiseid Microsofti konto või kohaliku konto lisamiseks.

[Lisateave kontode seadistamise kohta](https://support.microsoft.com/help/17197/)
