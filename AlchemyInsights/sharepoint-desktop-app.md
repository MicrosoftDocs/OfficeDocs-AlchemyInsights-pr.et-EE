# <a name="desktop-app-for-sharepoint"></a>Töölauarakendus SharePointi jaoks

Teie ettevõte on sisestatud SharePointi töölauarakenduse privaatse eelvaate programmi.

Microsoft ei toeta praegu töölauarakenduse rakendust SharePoint (privaatne eelvaade). Palun kasuta oma värvatud Microsoft Private Preview partnereid, et saada abi SharePointi töölauarakenduse jaoks.
