# <a name="put-apps-on-the-desktop"></a>Rakenduste asetama töölauale

Otsige rakendust menüüst **Start** ja Haarake rakenduse ikoon, hoides all vasakut hiirenuppu ja teisaldades ikooni töölauale.

Võite vajutada ka **Windowsi klahv + R** ja seejärel tippige see käsk:

`shell:appsfolder`

See avab **rakenduste** kausta **File Exploreris**. Nüüd võite lohistada rakenduse ikooni töölauale.
