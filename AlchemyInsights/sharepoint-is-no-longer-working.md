# <a name="sharepoint-is-no-longer-working"></a>SharePoint ei tööta enam

SharePointi töötamiseks võib olla mitu põhjust.

1. SharePointis võib ilmneda teenuse seisundiga seotud probleem. Lisateavet leiate teemast [Teenuse seisundi armatuurlaud.](https://admin.microsoft.com/AdminPortal/Home#/servicehealth)
2. Õigused võivad olla muutunud või eemaldatud. Lisateavet leiate teemast [SharePoint Online'ile ei pääse juurde.](https://docs.microsoft.com/sharepoint/troubleshoot/sharing-and-permissions/sharepoint-online-inaccessible)
3. Brauseriprobleemid võivad takistada kasutajatel SharePointi avamist. Paluge kasutajal proovida oma brauseri erarežiimi ja uuesti SharePointi sisse logida.
