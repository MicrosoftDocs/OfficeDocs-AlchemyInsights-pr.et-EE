# <a name="give-users-access-to-sharepoint-site"></a>Võimaldada kasutajatele juurdepääs SharePointi saidile

Juurdepääsu taotluse funktsioon võimaldab inimestel paluda luba sisu, mida nad on praegu õigus näha. Saidi omanikuna saate konfigureerida funktsiooni te meili saata, kui keegi taotleb juurdepääsu saidi. Lisateabe saamiseks külastage [seada üles ja Juurdepääsutaotluste haldamine](https://support.office.com/article/set-up-and-manage-access-requests-94b26e0b-2822-49d4-929a-8455698654b3).
