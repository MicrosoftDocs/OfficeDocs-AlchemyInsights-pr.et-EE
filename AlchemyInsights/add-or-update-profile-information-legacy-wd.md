# <a name="add-or-update-profile-information---legacy-wd---recommended-steps"></a>Profiili teabe lisamine või värskendamine – pärand WD – Soovitatavad toimingud

**Azure ' i arvelduse konto kontaktteabe muutmine**

**Maksja aadress**: Maksja aadress on ettevõtte või isiku aadress ja kontaktteave, kes vastutab arvelduse eest loodud arvete eest maksmise eest. [Microsoft Online ' i teenuse programmi (MOSP) arvelduse konto](https://docs.microsoft.com/azure/cost-management-billing/manage/change-azure-account-profile#update-an-mosp-billing-account-address)jaoks on olemas üks arve saaja aadress, mis kuvatakse kõigi konto jaoks loodud arvete korral.

**Teenuse ja turunduse e-posti aadress**: saate [määrata](https://docs.microsoft.com/azure/cost-management-billing/manage/change-azure-account-profile#change-your-contact-email-address) meiliaadressi, mis erineb meiliaadressist, mille sisse logite, et saada oma Azure ' i kontoga seotud olulisi arveldusega seotud, teenusega seotud ja soovitustega seotud teateid. Teenuse teatiste teatised (nt kiireloomulised turbega seotud probleemid, hindade muutmine või teie kontol kasutatavate teenuste muudatuste lõhkumine) saadetakse alati teie sisselogimise aadressile.

MOSP arvelduse konto aadressi värskendamiseks tehke järgmist.
1. Logige Azure ' i portaali sisse, kasutades meiliaadressi, millel on kontol konto administraatori õigused.
2. Otsige ja valige **kulude haldus + arveldus**. 
3. Klõpsake vasakus servas nuppu **Atribuudid** . 
4. Müüdud ja maksja aadresside värskendamiseks klõpsake nuppu **Uuenda arveldamise aadressi** . Sisestage uus aadress ja klõpsake nuppu **Salvesta**.

**Kontakti meiliaadressi muutmine** 

Kontakti meiliaadressi muutmiseks tehke järgmist.
1. Kui olete MOSP konto administraator, järgige juhiseid [MOSP arvelduse konto aadressi värskendamiseks](https://docs.microsoft.com/azure/cost-management-billing/manage/change-azure-account-profile#update-an-mosp-billing-account-address) ja klõpsake viimases etapis nuppu **Värskenda kontaktteave** . 
2. Sisestage uus meiliaadress. 
3. Liikuge Azure ' i portaali [kontaktteabe](https://ms.portal.azure.com/) paanile ja sisestage uus meiliaadress. 
4. Klõpsake ikooni, millel on teie initsiaalid või pilt. 
5. Valige **kontekstimenüü (...) > minu kontaktteave** ja sisestage uus meiliaadress.

**Profiili teabe lisamine või värskendamine**

Saate lisada kasutajaprofiili teavet (sh profiili pilti, tööalast teavet ja mõnda sätet) Azure Active Directory (Azure AD) abil. Lisateavet uute kasutajate lisamise kohta leiate teemast [kasutajate lisamine või kustutamine Azure Active Directorys](https://docs.microsoft.com/azure/active-directory/fundamentals/add-users-azure-active-directory).

Profiili teabe lisamiseks või muutmiseks tehke järgmist.

1. Logige sisse Azure ' i portaali kasutaja-administraatorina asutuse jaoks.
2. Valige **Azure Active Directory > kasutajad** ja seejärel klõpsake seda kasutajaprofiili, mida soovite värskendada. 
3. Iga saadaoleva jaotise teabe lisamiseks või värskendamiseks klõpsake nuppu **Redigeeri** . 
4. Klõpsake nuppu **Salvesta**.
