# <a name="solutions-for-error-30180-28"></a>Lahendused viga 30180-28

[Tõrge](https://support.office.com/article/47ae453b-677c-412f-9a21-6766555ff4de?wt.mc_id=Alchemy_ClientDIA) võib ilmneda 30180-28 Office'i värskenduse ajal kui teil on halb Interneti-ühendus. Siin on mõned asjad, mida võite proovida probleemi lahendada. 
  
- Taaskäivitage arvuti ning seejärel [Office'i värskenduse](https://support.office.com/article/2ab296f3-7f03-43a2-8e50-46de917611c5?wt.mc_id=Alchemy_ClientDIA).
    
- Järgige eespool kuid kasutage Office'i värskendamise ajal.
