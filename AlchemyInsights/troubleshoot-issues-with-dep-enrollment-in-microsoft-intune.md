# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>DEP-ga liitumise probleemide tõrkeotsing Microsoft Intune ' is

Vaadake allpool loetletud ressursid probleemi lahendamiseks kohe.
  
1. Kui DEP-seade ei saa registreeruda ja MFA (mitme teguriga autentimine) on lubatud, keelake MFA. Praegu ei toetata DEP registreerimise puhul MAKROMAJANDUSLIKKU

2. Levinud registreerimise tõrgete diagnoosimiseks ja lahendamiseks kasutage [tõrkeotsingu portaali Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) . Vaadake [seda dokumenti](https://docs.microsoft.com/intune/help-desk-operators) rohkem üksikasju.

3. Vaadake need dokumendid levinud tõrgete loendi, mis takistavad registreerimine ja lahendused iga: [tõrkeotsingu juhend](https://support.microsoft.com/help/4039809/troubleshooting-ios-device-enrollment-in-intune) ja [tõrkeotsing doc](https://docs.microsoft.com/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)

4. Teave [seadme registreerimise programmi kohta](https://docs.microsoft.com/intune/device-enrollment-program-enroll-ios).
