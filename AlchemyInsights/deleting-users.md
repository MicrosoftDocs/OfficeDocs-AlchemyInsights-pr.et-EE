# <a name="deleting-users"></a>Kasutajate kustutamine

Kui soovite kasutaja juurdepääsu kohe blokeerida ja peate oma teavet säilitama, vaadake palun [Blokeeri endise töötaja juurdepääs Microsoft 365 andmetele](https://docs.microsoft.com/microsoft-365/admin/add-users/remove-former-employee#block-a-former-employees-access-to-microsoft-365-data).
  
Kui soovite lihtsalt kustutada kasutaja ja ei ole seotud ühegi nende andmeid, vaadake palun [kustutage kasutaja oma organisatsiooni](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user). See annab kohe oma litsentsi vabaks.
