# <a name="access-denied-due-to-ip-restriction"></a>Juurdepääs on keelatud IP-piirangu tõttu

IT-administraatorina saate Microsoft 365 juurdepääsu SharePointi ja OneDrive ' i ressurssidele reguleerida, võttes aluseks määratletud võrgukohad, mida usaldate. Seda tuntakse ka asukoha-põhise poliitikana. Lisateavet leiate teemast [juurdepääsu reguleerimine võrgu asukoha](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location)alusel.
