# <a name="diagnostic-to-enable-basic-authentication-for-exchange-online-protocols"></a>Diagnostika protokollide jaoks elementaarse autentimise Exchange Online lubamiseks

Selle diagnostika abil saate lubada Exchange Online-protokollide (nt POP3, IMAP4, Exchange ActiveSync, Exchange Veebiteenused, Ühenduseta aadressiraamat, MAPI, RPC ja Remote PowerShell) põhiautentimise, mille Microsoft on hiljuti teie asutuse jaoks keelanud. 

Saadame sõnumikeskuse kaudu otseteadmist, et rentnikud teaksid, kus nad on valmis oma keskkonnas elementaarse autentimise välja lülitama, kuna see aitab kaitsta nende keskkonda seotud turberiskide eest.
