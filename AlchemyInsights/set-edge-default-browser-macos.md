# <a name="set-microsoft-edge-as-the-default-browser-on-a-macos-device"></a>Microsoft Edge'i seadis macOS-seadme vaikebrauseriks

Microsoft Edge'i vaikebrauseriks vaatamiseks kasutage ühte kahest meetodist.

1. meetod. Seadme vilkumine macOS-i pildiga, kus Microsoft Edge on juba vaikebrauseriks määratud.

2. meetod. Määrake poliitika DefaultBrowserSettingEnabled ( VaikebrowserSettingEnabled), et paluda kasutajal seada Microsoft Edge vaikebrauseriks.

Kumbki meetod võimaldab kasutajal vaikebrauseri muuta. Seetõttu soovitame juurutada poliitika DefaultBrowserSettingEnabled isegi siis, kui kasutasite meetodit 1. Kui kasutaja muudab pärast poliitika juurutamist vaikebrauseri, palub poliitika kasutajal seada vaikebrauseriks tagasi Microsoft Edge'i.
