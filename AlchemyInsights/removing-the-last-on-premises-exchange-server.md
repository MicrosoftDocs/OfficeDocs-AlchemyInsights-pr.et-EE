# <a name="removing-the-last-on-premises-exchange-server"></a>Viimase asutusesisese Exchange Serveri eemaldamine

Enne eemaldamist viimase Exchange Serveri oma asutusesiseses organisatsioonis, veenduge, et mõistate selleks, et otsustada, kui peate selle viimati server muudel eesmärkidel. Lisateabe saamiseks vaadake [Kuidas ja millal teie asutusesisese Exchange'i serverid kooseksisteerimise demonteerimise](https://technet.microsoft.com/en-us/library/dn931280%28v=exchg.150%29.aspx).
