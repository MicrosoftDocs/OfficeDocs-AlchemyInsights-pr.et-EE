# <a name="upload-a-microsoft-teams-meeting-recording-to-stream-a1"></a>Microsoft Teamsi koosoleku salvestamise voogesituseks üleslaadimine (a1)

Enamiku Microsoft 365 kasutajate jaoks laaditakse meeskonnad koosolekul automaatselt üles Microsoft Streami. Kuid alates  **augustist 20, 2020**, kui kasutajad, kellel on a1 (sh a1 +) litsentsid, alustavad teamsi koosoleku salvestamist, muutub see käitumine.  

Lisateavet leiate teemast [Microsoft teamsi koosoleku salvestamise voogesituse üleslaadimine](https://docs.microsoft.com/stream/portal-upload-teams-meeting-recording).
