# <a name="workflows-in-sharepoint"></a>SharePointi töövood

<p>SharePointi töövood on eelprogrammeeritud minirakendusi, mis sujuvamaks ja automatiseerida erinevaid äriprotsesse. Töövood võivad ulatuda koguda allkirju, tagasiside või tunnustamise kava või dokumendi, rutiinne menetlus praeguse oleku jälgimine.</p> <p><strong>Alustamine:</strong> Kui sa&rsquo;oled just alustanud SharePointi töövoogudega, vt <a href="https://support.office.com/en-us/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3">Sissejuhatus SharePoint töövood</a> rohkem infot.</p> <p><strong>Microsoft voolu:</strong> Võite kasutada Microsofti Flow klassikaline SharePointi töövoo lahenduste asemel. Lisateavet leiate teemast <a href="https://support.office.com/en-us/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01">loomine voolu loendi või teegi SharePoint Online'i või OneDrive</a>.</p> <p><strong>Tõrkeotsingu ja silumise töövoo:</strong> Silumine töövood meenutab eelmise versiooni. Lisateavet vt <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/debugging-sharepoint-server-workflows">SharePointi silumine töövood</a>. Tõrkeotsingu kohta leiate teemast <a title="tõrkeotsing SharePoint Server töövoo valideerimisvigu" href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio">Töövoo SharePoint Server valideerimistõrgete tõrkeotsing</a>.&nbsp;</p>
