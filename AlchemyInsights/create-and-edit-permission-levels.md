# <a name="working-with-sharepoint-permission-levels"></a>Töötamine SharePointi õiguste tasemetel

SharePointis on olemas vaikimisi õigusetasemed, mida saate kasutada (nt **panusta** ja **Kuva ainult**). Kuid kui üks neist ei vasta teie vajadustele, saate luua uue õigusetaseme. Pidage meeles, et õiguste tasemete jälgimine on lihtsam, kui te vaikesätteid ei muuda.
