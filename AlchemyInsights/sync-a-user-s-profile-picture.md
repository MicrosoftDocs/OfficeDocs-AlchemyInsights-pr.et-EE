# <a name="sync-a-users-profile-picture"></a>Sünkroonida kasutaja profiili pilt

Profiilisünkroonimise pilt SharePoint Online'i kasutaja nõuab oma foto sünkroonimise algatada. Ei ole automaatse sünkroonimise SharePoint Online fotosid. SharePoint Online'i sünkroonib ainult fotod: Exchange Online, Azure Active Directory.
  
Kui kasutaja kunagi toimingu tõttu sünkroonimise küsivaid või neid uuendada oma profiili pilti ja uus üks ei ilmu SharePoint Online, peate küsima sünkroonib kasutaja. Siin on, kuidas seda on võimalik teha:
  
1. Avage veebibrauser, tolli-ja erasektori või Incognito session ja [minna Office.com](https://na01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.office.com%2F&amp;data=02%7C01%7C%7Cbad62c504a36446096e108d614dec653%7C72f988bf86f141af91ab2d7cd011db47%7C1%7C0%7C636719344369977864&amp;sdata=Km7ZnN8FHSouZbxOiEpQAGIKsK82SHr25uYCh3Gc%2F3s%3D&amp;reserved=0).
  
2. Logige sisse oma töö või kooli kontole ja klõpsake paani OneDrive'i või SharePointi.
