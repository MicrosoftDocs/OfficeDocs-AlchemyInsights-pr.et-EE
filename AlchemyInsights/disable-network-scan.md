# <a name="disable-network-scan"></a>Võrgu skannimine keelamine

Võrgu ühiskasutuse skaneerimine võib mõjutada jõudlust.  Kui soovite tagada, et klient ei kontrolliks vaikimisi võrgu aktsiaid/faile, konfigureerige Windows Defenderi rakenduses **True** järgmised sätted.

- DisableScanningMappedNetworkDrivesForFullScan
- DisableScanningNetworkFiles
