# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Välissuunalisele vormile ei saa manust lisada

Kui proovite lisada manust välisele Microsoft Formsi uuringule, pole see funktsioon veel saadaval. Selle funktsioonitaotluse oleku jälgimiseks [sellele UserVoice'i kirjele](https://go.microsoft.com/fwlink/?linkid=2133069)ja lisage oma hääl.
