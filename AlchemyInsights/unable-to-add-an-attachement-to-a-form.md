# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Väliselt suunatud vormile ei saa manust lisada

Kui proovite lisada manuse välisele Microsoft Formsi küsitlusele, pole see funktsioon veel saadaval. Selle funktsiooni oleku jälgimiseks liikuge [sellele UserVoice kirjele](https://go.microsoft.com/fwlink/?linkid=2133069)ja lisage oma hääl.
