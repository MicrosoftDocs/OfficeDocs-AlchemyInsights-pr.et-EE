# <a name="already-paid"></a>Juba makstud?

Kui olete juba makse saatnud, võite meile teada:
  
1. administreerimiskeskuses lehele **arveldamise** \> [tellimused](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Valige tellimus ja valige jaotisest **minu arve** käsk **Kuva Bill**.

3. Kas valida lehel **Arve üksikasjad** **juba makstud?** Teil palutakse sisestada saadetud maksega seotud teave.

4. Kui kogu vajalik teave on sisestatud, valige **saada**.
