# <a name="report-suspicious-urls"></a>Kahtlaste URL-ide teatamine

Kahtlaste URL-ide esitamiseks tehke järgmist.

1. Sisestage **uue edastuse** HÜPIK täielik URL (sh protokoll) (https://).
2. Kui valite oleks **pidanud olema blokeeritud**, määrake, kas URL on **andmepüügi** või **pahavara**.
3. Valige **Edasta**.
