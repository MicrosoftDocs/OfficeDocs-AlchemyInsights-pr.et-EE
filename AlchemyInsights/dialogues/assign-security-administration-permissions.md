# <a name="assign-the-security-administration-permissions"></a>Turbe halduse lubade määramine

1. Valige Office 365 turbe & täitmise keskuses **õigused > Security Administrator**.
2. Klõpsake paanil **Security Administrator** **liikmete** kõrval nuppu **Redigeeri**.
3. Klõpsake paanil **Redigeeri liikmete valimine** käsku **Vali liikmed**.
4. Klõpsake paani **liikmete valimine** nuppu **+ Lisa** ja seejärel valige kasutajate nimed.
5. Valige **lisa > valmis > Salvesta**.
