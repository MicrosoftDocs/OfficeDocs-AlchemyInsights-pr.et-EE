# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Microsoft Intune ' i kasutamine Microsoft Edge ' i juurutamiseks macOS-seadmes

Rakenduse juurutamiseks, konfigureerimiseks, jälgimiseks ja kaitsmiseks macOS-seadmes peate rakenduse lisama Microsoft Intune ' i. Rakendus peab olema Intune ' i toetatud tüübist ja Intune ' is saadaolevad halduse suvandid sõltuvad rakenduse tüübist. Intune ' i kasutamine tähendab ka seda, et te ei pea kasutama macOS-i rakenduste pakkimise tööriista.

Intune toetab Microsoft Edge ' i. Lisateavet leiate teemast Microsoft [Edge ' i lisamine macOS-i seadmetele Microsoft Intune](https://go.microsoft.com/fwlink/?linkid=2134949)' i abil.
