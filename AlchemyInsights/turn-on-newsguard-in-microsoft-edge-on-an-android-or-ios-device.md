# <a name="turn-on-newsguard-in-microsoft-edge-on-an-android-or-ios-device"></a>Microsoft Edge ' i NewsGuard sisselülitamine Androidi või iOS-seadmes

NewsGuard on kolmanda osapoole laiend, mis hindab ja hindab veebisaite USA usaldusväärsuse ja läbipaistvuse ajakirjanduslike standardite osas. See laiend on saadaval Microsoft Edge ' is, kuid see on vaikimisi välja lülitatud. Kui funktsioon on sisse lülitatud, siis laiend ei takista juurdepääsu saitidele ega Katkesta sirvimist.

NewsGuard sisselülitamiseks tehke järgmist.
1. Valige... > **Sätted**  >  **Uudiste hinnang**.
2. Lülitage **aadressiribal kuvatav hinnang**.
