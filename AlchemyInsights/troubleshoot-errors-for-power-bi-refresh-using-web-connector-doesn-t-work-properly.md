# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Värskendamine veebi konnektori abil ei tööta korralikult

Kui teil on Web Connector skripti, mis kasutab funktsiooni [Web. Page](https://msdn.microsoft.com/library/mt260924.aspx) ja olete uuendanud oma DataSet või aruande pärast november 18th, 2016, peate kasutama lüüsi, et värskendada korralikult töötada.
