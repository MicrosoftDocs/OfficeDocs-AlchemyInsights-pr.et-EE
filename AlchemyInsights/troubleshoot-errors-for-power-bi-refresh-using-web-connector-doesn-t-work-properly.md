# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Värskendamine veebi konnektori abil ei tööta õigesti

Kui teil on veebikonnektori skript, mis kasutab funktsiooni [Web. Page](https://msdn.microsoft.com/library/mt260924.aspx) , ja olete oma andmekomplekti või aruandeid uuendanud pärast 18. novembrit, 2016, peate värskendamiseks kasutama lüüsi, et õigesti töötada.

Lisateavet leiate järgmistest teemadest. [https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
