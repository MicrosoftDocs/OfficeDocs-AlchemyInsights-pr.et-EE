# <a name="turn-off-focused-inbox-in-email-clients"></a>Lülitada sisse keskendunud meiliklientide

Suunatud postkasti Outlook 2016, Outlook Web väljalülitamiseks Outlook.com/Hotmail.com, Outlook 2016 for Mac või Mail for Windows 10, vt [keskendunud sisse lülitada](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).
  
Vaata kõigile organisatsiooni keskendunud sisendkausta väljalülitamiseks [Konfigureerida keskendunud sisendkausta kõigile teie ettevõttes](https://support.office.com/article/613a845c-4b71-41de-b331-acdcf5b6625d.aspx).
