# <a name="app-configuration-policies-for-microsoft-intune"></a>Microsoft Intune ' i rakenduse konfigureerimise põhimõtted

Microsoft Intune ' i rakenduse konfigureerimise poliitikad pakuvad iOS-i või Androidi rakenduste jaoks konfiguratsioonisätteid. Need konfiguratsiooniseaded võimaldavad rakendust kohandada. Konfigureerimise poliitika tugi on operatsioonisüsteemi versioonide ja rakenduste jaoks piiratud.

Lisateavet leiate teemast [Microsofti Intune ' i rakenduse konfigureerimise põhimõtted](https://docs.microsoft.com/intune/app-configuration-policies-overview).
