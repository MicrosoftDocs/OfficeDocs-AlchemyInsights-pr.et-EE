# <a name="onedrive-cant-add-your-folder-right-now"></a>OneDrive ei saa praegu oma kausta lisada

Tõrge "Kahjuks ei saa OneDrive teie kausta praegu lisada", kui  teil on teeki,  kus on veerud Väljaregistreerimine või Valideerimine, või kui mustandüksuse turve on seatud väärtusele Ainult kasutajad, kes saavad redigeerida, või Ainult kasutajad, kes saavad teegi versiooni Sätted üksusi kinnitada.    

Juhised sünkroonimise blokeerimise veerusätete eemaldamiseks leiate teemast Piirangud ja piirangud OneDrive [ja SharePoint](https://support.microsoft.com/office/64883a5d-228e-48f5-b3d2-eb39e07630fa).
