# <a name="move-a-folder-in-outlookcom"></a>Teisalda kaust Outlook.com

Saate kaustu **lemmikute**all ümber paigutada, kuid kaustu ei saa **kaustadesse** ümber paigutada ega kausta alamkausta teha. Kui teisaldate kogemata kausta mõnda muusse kausta, looge uus kaust ja kopeerige kõik oma meilisõnumid alamkaustast uude kausta.
  
Lisateabe saamiseks lugege [Outlook.com sõnumikaustadega töötamise](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy)kohta.
