# <a name="solutions-for-unsupported-operating-system"></a>Lahendused "toetuseta operatsioonisüsteem"

Kui proovite installida Microsoft 365, võidakse kuvada **toetuseta operatsioonisüsteemi** tõrketeade, kui seade ei vasta [süsteeminõuetele](https://products.office.com/office-system-requirements).
  
Microsoft 365 nõuab:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 või Windows Server 2008 R2

    Või

- Mac OS X 10,10

Kui saite selle sõnumi Office ' i mobiilsideseadme või Chromebooki installimise ajal, vaadake, [Kuidas installida ja käivitada Microsoft Office ' i Chromebookis](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
