# <a name="solutions-for-unsupported-operating-system"></a>"Toetuseta operatsioonisüsteem" lahendusi



Kui proovite installida Office 365, võib ilmneda **muu opsüsteem** veateate kui seade ei vasta [süsteeminõuetele](https://products.office.com/office-system-requirements).
  
Office 365 nõuab:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 või Windows Server 2008 R2
    
    VÕI
    
- Mac OS X 10.10
