# <a name="add-permissions"></a>Õiguse lisamine

1. Administraatori nõusoleku lõpp-punkti abil saate [konfigureerida rakenduse staatiliselt nõutud õigustega loendi](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. Saadaval on kaks õigust, mille abil saab luua rakenduse registreerimisi, millel on erinev käitumine. Lisateavet leiate teemast [rakenduse Azure Active Directory kohandatud rollide rakenduse registreerimise load](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions).
1. [Rollide permissions](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): tabelid, mis kirjeldavad Azure Active Directory teatud õiguseid, mis on antud igale rollile. Mõnel rollil võib olla Microsofti teenustes väljaspool Azure Active Directory täiendavaid õiguseid.
