# <a name="dashboard-doesnt-reflect-changes-after-refresh"></a>Armatuurlaud ei kajasta muutusi pärast värskendamist

Kui armatuurlaud ei kajasta muutusi pärast värskendamist, oodake umbes 10-15 minutit värskendamiseks armatuurlaua paanil. Kui seda ikka ei kuvata, kinnitage visualiseeringu uuesti armatuurlauale.

Lisateavet leiate teemast [värskendamise stsenaariumide tõrkeotsing](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios).
