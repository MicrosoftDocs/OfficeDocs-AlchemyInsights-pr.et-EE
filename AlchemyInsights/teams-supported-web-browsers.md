# <a name="teams-supported-web-browsers"></a>Teamsi toetatud veebibrauserid

Teamsi toetab täielikult [Microsoft teamsi artiklis, piirangutes ja tehnilistes kirjeldustes](https://docs.microsoft.com/microsoftteams/limits-specifications-teams#browsers)loetletud Interneti-brausereid, kus on märgitud kõnede ja koosolekute erandid.
