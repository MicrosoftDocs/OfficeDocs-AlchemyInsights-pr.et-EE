# <a name="edit-billing-profile-for-apps"></a>Rakenduste arvelduse profiili redigeerimine

## <a name="to-change-the-billing-profile-on-apps"></a>Arvelduse profiili muutmine rakendustes

1. Avage leht **Arveldamine**  >  **[oma tooted](https://go.microsoft.com/fwlink/p/?linkid=842054)** ja seejärel valige vahekaart **rakendused** .

2. Otsige üles ja valige rakendus, mida soovite hallata.  

3. Valige jaotises **arvelduse profiil**nupp **Redigeeri** ja valige arveldamise profiil, mida soovite rakendusega seostada.

## <a name="edit-billing-profiles"></a>Arvelduse profiilide redigeerimine

1. Avage leht **arvelduste**  >  **[arved & maksed](https://go.microsoft.com/fwlink/p/?linkid=848039)** ja seejärel valige **arveldamise profiilid**.

2. Valige redigeeritav loendist arvelduse profiil.
