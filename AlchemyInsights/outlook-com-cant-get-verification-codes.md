# <a name="cant-get-verification-codes"></a>Ei saa kinnituskoodid

Kui te ei pääse oma alternatiivsele meiliaadressile või telefoninumbrile, minge Microsofti konto [turvalisuse põhitõed](https://account.microsoft.com/security) lehele, valige **mul pole ühtegi neist** lehele Sisselogimine ja järgige turbeteabe asendamine.

*Märkus:* [Rakendusega Microsoft Authenticator](https://go.microsoft.com/fwlink/?linkid=2016117) ei nõua ühendatud teile kinnituskoodi.

Lisateavet lugeda [Microsofti konto info ja kontrolli turbekoode](https://support.microsoft.com/help/12428/).
