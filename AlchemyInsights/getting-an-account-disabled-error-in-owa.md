# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Kas Outlooki veebirakenduses on kontoga keelatud tõrge?

Kui ilmneb tõrge, mille **teie konto on keelanud**  `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` , kui proovite avada Outlooki veebirakendust (varem tuntud kui Outlook Web App), võib teie ettevõtte administraator keelata teie juurdepääsu Outlooki veebirakenduses. Teie administraator saab selle tõrke lahendada, järgides teemas rakenduse [Outlook Web App lubamine või keelamine postkastis](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx)toodud juhiseid.
