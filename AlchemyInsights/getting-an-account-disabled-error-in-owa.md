# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Outlooki veebis tõrge konto keelamise kohta?

Kui teil tekib tõrge, et **teie konto on keelatud** , `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` kui proovite avada Outlook Web (varem tuntud kui Outlook Web App), teie organisatsiooni admin võib olla keelanud teie juurdepääsu Outlook Web. Teie administraator saab selle tõrke lahendada, järgides juhiseid teemas [lubada või keelata Outlook Web Appi postkasti](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
