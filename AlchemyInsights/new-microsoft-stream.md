# <a name="microsoft-stream"></a>Microsoft Stream

Microsoft Stream on ettevõtte video teenus, kus teie asutuse töötajad saavad videoid üles laadida, vaadata ja ühiskasutusse anda. Saate ühiskasutusse anda klasside, koosolekute, esitluste, koolituskursuste või muude videote salvestisi, mis aitavad teie meeskonna koostööd teha.

- [Microsoft stream (klassikaline)](https://docs.microsoft.com/stream/overview)
- [Microsoft Streami (Classic) migreerimine uude Microsoft Streami](https://docs.microsoft.com/stream/classic-migration)
