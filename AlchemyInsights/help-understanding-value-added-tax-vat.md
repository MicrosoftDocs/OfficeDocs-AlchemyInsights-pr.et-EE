# <a name="help-understanding-value-added-tax-vat"></a>Käibemaksu mõistmine (km)

Kui olete riigis, mis kasutab väärtustatud käibemaksu (km), näiteks Ühendkuningriik, arvutab Microsoft arveldamine teie eest käibemaksu ja rakendab summa teie arvetele.
  
Maksukaalutlustel tuleb teil määrata ka käibemaksu registreerimisnumber Arveldamissätetes.
  
Käibemaksumäärasid ei saa lisada ega eemaldada, kuid saate muuta vaikimisi määratud käibemaksumäära sätetes. Vaikimisi kasutab arveldamine käibemaksu registreerimisnumbrite kinnitamiseks teenust, kuid soovitame lugeda Euroopa komisjoni avaldatud VIES-i [lahtiütlemise klauslit](https://go.microsoft.com/fwlink/?LinkID=841741) . Kui otsustate teenuse keelata, peate kontrollima kõiki KÄIBEMAKSUKOHUSTUSLASE registreerimisnumbreid.
