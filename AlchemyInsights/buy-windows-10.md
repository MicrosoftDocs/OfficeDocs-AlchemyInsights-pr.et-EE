# <a name="how-to-buy-a-copy-of-windows-10"></a>Kuidas osta koopia Windows 10

Ajakohane hind ning selle kohta, kuidas osta ja alla laadida Windows 10, avage [Microsoft Store](https://www.microsoft.com/store/b/windows).

**Märkus.** Windows 10 tasuta versioonitäiendus Hangi Windows 10 (GWX) kaudu lõppes 29. juulil 2016.

**Tähtis.** Kui kasutate versiooni Windows 7, soovitab Microsoft liikuda uude arvutisse, kus töötab Windows 10, kuna Windows 7 saavutas 14. jaanuaril 2020 tugiteenuste elutsükli lõpu.
