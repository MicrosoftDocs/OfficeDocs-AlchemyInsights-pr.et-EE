# <a name="map-a-sharepoint-library-to-a-network-drive"></a>SharePointi teegi vastendada võrgudraivile

Teegi vastendamine võrgudraiviga on ajutine ja seda toetatakse ainult Internet Exploreri kaudu. Selle asemel [sünkroonida SharePointi failid uue OneDrive ' i Sünkroonimisklient](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx), mis pakub [faile nõudmisel](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Juurdepääs kõigile failidele OneDrive ilma kohalikku salvestusruumi.
