# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Microsoft Edge ' i lõpp-punktide lubamise loendi loomine

Kui soovite, et teie käsutuses oleks Microsoft Edge ' i funktsioonide komplekt, peab see olema Internetiga ühendatud. Lisaks sellele, et lubada brauseril suhelda tulemüüride ja muude Interneti-turbe mehhanismide kaudu, peate lisama domeeni URL-ide kogumi, et [lubada selle lõpp-punktide loend](https://go.microsoft.com/fwlink/?linkid=2135054).
