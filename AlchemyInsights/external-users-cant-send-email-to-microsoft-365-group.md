# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Välised kasutajad ei saa saata meilisõnumeid Microsoft 365 rühmale

Nende juhiste abil saate lubada väliskasutajatel meilida Microsoft 365 rühma.

1. Logige sisse [administreerimiskeskuse](https://admin.microsoft.com/)  >  **rühmade**  >  **rühmadesse**.
2. Klõpsake paani sätted avamiseks rühma, mida soovite hallata.
3. Märkige vahekaardil **sätted** ruut **Luba välised saatjad sellele rühmale meilisõnumite**saatmiseks.
4. Valige **Salvesta**.</br>
    Sätete kajastamiseks saate lubada kuni tund.
