# <a name="setting-up-a-hybrid-environment"></a>Hübriid-keskkonna häälestamine

Kui soovite, et kasutajad saaksid sujuvalt koos töötada, olenemata sellest, kas nende andmeid talletatakse kohapeal serverites või Microsoft 365, peate võib-olla häälestama hübriid-keskkonna. Võimaluste ülevaate saamiseks vaadake [Microsoft 365 integreeritust kohapealsete keskkondade abil](https://docs.microsoft.com/office365/enterprise/office-365-integration).
