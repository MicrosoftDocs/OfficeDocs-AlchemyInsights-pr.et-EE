# <a name="fix-problems-uploading-files-to-onedrive"></a>OneDrive ' i failide üleslaadimise probleemide lahendamine

Proovige seda, kui faili ei laadita üles:
  
- Nimetage fail ümber ja proovige uuesti. Mõned failinimed ja failinimed ei ole lubatud. 
    
- Kui sünkroonite faili, teisaldage see oma OneDrive ' i kaustast välja (näiteks teisaldage see töölauale) ja kui teie OneDrive ' i sünkroonimise olek kuvatakse kui "ajakohane", teisaldage see tagasi.
