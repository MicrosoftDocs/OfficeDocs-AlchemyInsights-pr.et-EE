# <a name="fix-problems-uploading-files-to-onedrive"></a>OneDrive ' i failide üleslaadimisega seotud probleemide lahendamine

Proovige seda, kui faili ei laadita üles:
  
- Nimetage failid ümber ja proovige uuesti. Failinimed ja tähemärgid failinimedes pole lubatud. 
    
- Kui sünkroonite fail, teisaldage see oma OneDrive ' i kaustast (nt teisaldage see töölauale) ja kui teie OneDrive ' i sünkroonimise olek kuvatakse kujul "ajakohane", viige see tagasi.
