2. <span data-ttu-id="e6490-p102">**Välised kasutajad väliste gruppi** - välised rühmad väliskasutajatele. Väliste kasutajate juurdepääsemiseks on ainult välise kontrollrühma. 3. <span data-ttu-id="e6490-p103">**Väliste kasutajate Yammeri vestlust sisemine rühmas** - see valik võimaldab lisada mõne kindla vestluse osalejatega. 4. <span data-ttu-id="e6490-p104">**Välise võrgustikud** - see valik võimaldab kutsuda väliskasutajatele välise võrgu, mille administraatorid saate luua oma võrku.
