# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>Puhverserveri aadressi tõrge ühiskasutatava postkasti loomisel

Kui proovisite luua ühiskasutuses postkasti ja saite selle tõrketeate "puhverserveri aadress" SMTP:alias@domain.com "on juba kasutusel...", on teie valitud meiliaadress juba teie asutuse teine postkast.
  
Peate leidma kasutaja, rühma või ühiskasutusega postkasti, millel on see meiliaadress, ja selle kustutama või muutma selle meiliaadressi. Seejärel saate luua ühiskasutusega postkasti, kus on vaba meiliaadress. Selle leidmiseks kasutage avalehel otsingut.
  
Kui te ei soovi olemasolevat meiliaadressi kustutada, valige oma ühiskasutatava postkasti jaoks uus meiliaadress.
