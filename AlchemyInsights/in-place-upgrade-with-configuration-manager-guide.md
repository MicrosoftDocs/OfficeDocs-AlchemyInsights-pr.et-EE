# <a name="in-place-upgrade-with-configuration-manager-guide"></a>Kohapealne täiendus Configuration Manageri juhendiga

Kohapealne versiooniuuendus säilitab olemasoleva operatsioonisüsteemi versiooni kõik andmed, sätted, rakendused ja draiverid. See on loodud äärmiselt usaldusväärseks ning tal on võimalus automaatselt eelmise operatsioonisüsteemi juurde tagasi pöörduda, kui ilmnevad probleemid.

Windows 7 ja Windows 8,1 seadmete täiendamiseks Windows 10 uusimale versioonile kasutage [kohapealset versioonitäiendust Configuration Manageri juhendiga](https://admin.microsoft.com/adminportal/home#/win10upgrade) . Eeltingimuste kontrollimiseks ja kohapealse täienduse automaatseks konfigureerimiseks kasutage ette nähtud skripti.
