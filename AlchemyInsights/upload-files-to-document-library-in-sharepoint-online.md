# <a name="upload-files-to-onedrive-or-sharepoint"></a>Failide üleslaadimine OneDrive ' i või SharePointile

Saate faile dokumenditeeki SharePoint Online üles laadida, lihtsalt lohistades neid oma arvutist ja kukutades need dokumenditeeki. OneDrive ' iga saate sünkroonida faile arvuti ja pilve vahel, et saaksite oma failidele kõikjalt juurde pääseda. Samuti saate oma sünkroonitud failidega töötada otse File Exploreris ja pääseda juurde failidele ka siis, kui olete võrguühenduseta. Lisateavet üleslaadimise ja sünkroonimise kohta leiate alltoodud linkidest.
