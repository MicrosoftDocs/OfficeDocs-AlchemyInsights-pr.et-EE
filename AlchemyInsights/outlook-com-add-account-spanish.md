# <a name="agregar-sus-otras-cuentas-de-correo-electrnico"></a>Agregar sus allapoole Cuentase de correo electrónico

Cuando agregue allapoole Cuentase de correo electrónico Outlook.com, puede usar Outlook.com para enviar y leer mensajes de correo electrónico de estas Cuentase sin salir de Outlook.com-i. Cada cuenta que se agrega Outlook.com se denomina una cuenta conectada. También puede [crear ÜRO alias de correo electrónico](https://support.office.com/es-es/article/agregar-o-quitar-un-alias-de-correo-electrónico-en-outlook-com-459b1989-356d-40fa-a689-8f285b13f1f2) que utiliza la misma Bandeja de entrada, enviar, elementos y lista de contactos como su cuenta de Outlook.com-i existente.

[Lea cómo agregar sus allapoole Cuentase de correo electrónico on Outlook.com.](https://support.office.com/es-es/article/agregar-sus-otras-cuentas-de-correo-electrónico-a-outlook-com-c5224df4-5885-4e79-91ba-523aa743f0ba?ui=es-ES&rs=es-ES&ad=ES)
