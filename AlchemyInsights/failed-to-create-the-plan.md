# <a name="failed-to-create-the-plan-error-when-trying-to-create-a-planner-tab-in-microsoft-teams"></a>Tõrge "Plaani loomine nurjus" Microsoft Teamsis Planneri vahekaardi loomisel

Kui teile kuvatakse Microsoft Teamsis Planneri vahekaardi loomisel tõrge "Plaani loomine nurjus", veenduge, et administraator on rühma liige, järgides järgmisi juhiseid.

1. Avage halduskeskuses **Rühmarühmad**  >  [](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/groups). 
2. Valige rühm ja seejärel valige vahekaardil **Liikmed** nupp Kuva **kõik ja halda liikmeid.**
3. Lisage liige ja valige **Salvesta**.
