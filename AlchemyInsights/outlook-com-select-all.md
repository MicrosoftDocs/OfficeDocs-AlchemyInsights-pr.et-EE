# <a name="how-to-select-all-in-outlookcom"></a>Kuidas "Vali kõik" in Outlook.com

Saate valida ainult siis, kui kõik sõnumid on nähtavad. See on vältida juhuslikke toiminguid sõnumeid, mida te ei näe.

**Ctrl + kliki** valib mitu eraldi sõnumit.

**SHIFT + klõpsu** valib kõik sõnumid valitud esimese ja viimase sõnumi vahel.
