# <a name="how-to-select-all-in-outlookcom"></a>Kuidas "Vali kõik" Outlook.com-is

Kõik saate valida ainult siis, kui kõik on nähtav. See on takistada ei näe sõnumeid juhusliku meetmetes.

**CTRL + klõps** valib mitu üksikuid kirju.

**SHIFT + klõpsake** kõik sõnumi vahel valite esimese ja viimase sõnumi valimine.
