# <a name="how-do-i-add-a-credit-card"></a>Kuidas lisada krediitkaarti?

1. Administreerimiskeskuses, [tellimuste](https://go.microsoft.com/fwlink/p/?linkid=842054) lehele või valida **arvete** \> **tellimused**.
    
2. Valige tellimus, mida soovite uue krediitkaardi või pangakonto maksta ja valige **värskendus makseandmeid**.
    
3. Valige rippmenüü noolt all **Lisa või valige makseviisi**, seejärel **uue kaardi** või **uue pangakonto**.
    
4. Uus krediitkaart või pangakonto kohta käiv teave ja valige Esita.
