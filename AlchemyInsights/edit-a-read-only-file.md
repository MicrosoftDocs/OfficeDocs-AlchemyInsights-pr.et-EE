# <a name="edit-a-read-only-file"></a>Kirjutuskaitstud faili redigeerimine

Dokumenditeegis, kui klõpsate dokumendi nime, võib Office ' i programm selle ootamatult avada kirjutuskaitstud failina. Redigeerimiseks avamiseks klõpsake nuppu **Ava** (kolm punkti) ja seejärel klõpsake nuppu **Redigeeri.**
