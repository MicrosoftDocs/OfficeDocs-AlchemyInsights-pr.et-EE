# <a name="edit-a-read-only-file"></a>Kirjutuskaitstud faili muutmine

Dokumenditeekides, klõpsate dokumendi nimi, Office'i programm võib ootamatult avada kirjutuskaitstud failina. Redigeerimiseks avada, klõpsake **avatud** (kolm punkti) ja seejärel klõpsake nuppu **Muuda**
