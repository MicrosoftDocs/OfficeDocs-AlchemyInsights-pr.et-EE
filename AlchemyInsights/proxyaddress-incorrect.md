# <a name="proxyaddress-incorrect"></a>ProxyAddress on vale

Kui objekt sünkroonitakse Azure AD-ga, võrreldakse Active Directory atribuuti proxyAddresses määratud väärtusi Azure AD reeglitega ja seejärel on atribuut proxyAddresses Azure AD asustatud. Seetõttu ei pruugi Active Directorys objekti atribuudi proxyAddresses väärtused olla samad, mis Azure AD proxyAddresses atribuudi väärtus.
  
Lisateavet proxyAddress asustamise kohta leiate teemast [proxyAddress atribuudi asustamine AZURE ad-s](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
