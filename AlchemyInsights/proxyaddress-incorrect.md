# <a name="proxyaddress-incorrect"></a>Kasutajaatribuut vale

Kui objekt on sünkroonitud Azure AD, väärtused, mis on määratud atribuudis proxyAddresses Active Directory on Azure AD eeskirjad võrreldes ja seejärel atribuuti proxyAddresses on asustatud Azure AD. Seetõttu Active Directory objekti atribuuti proxyAddresses väärtused ei pruugi olla sama atribuuti proxyAddresses väärtused Azure AD.
  
Lisateavet kuidas on Kasutajaatribuut on asustatud, Vaata [Kuidas Kasutajaatribuut atribuut on asustatud Azure AD](https://support.microsoft.com/en-us/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
