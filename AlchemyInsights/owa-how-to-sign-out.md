# <a name="how-to-sign-out-of-outlook-on-the-web"></a>Kuidas Outlook Web väljalogimine

1. Valige oma nimel või pildil üleval paremal nurgas.
    
    Ekraani ülaosas oma pilti ei kuvata, siis kontrollige, kui reklaami blokeerimine on sisse lülitatud. Mõnikord see takistab pildi näitamist.
    
2. Valige **Logi välja**. 
    
    Peata automaatselt sisselogimine töö või kooli kontole, tühjendage märkeruut jaoks **Jäta mind meelde** lehel, kui sisestate parooli. 
    
Lisateavet leiate teemast [sisselogimine või Outlook veebis kokku](https://support.office.com/article/763fab4d-0138-4814-b450-37fc286bcb79).
