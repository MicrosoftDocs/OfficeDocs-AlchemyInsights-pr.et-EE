# <a name="error-onedrive-cannot-connect-to-windows"></a>Tõrge: OneDrive ei saa Windowsiga ühendust luua

Tõrketeate lahendamiseks otsige üles cldflt.sys fail, et näha, kas see töötab, ja kui pole, siis lubage see. 
  
Lisateavet cldflt.sys-faili lubamise kohta leiate teemast OneDrive ' i [failidele juurdepääsemisel ilmneb tõrge OneDrive ' i kaudu ei saa ühendust luua Windowsiga.](https://go.microsoft.com/fwlink/?Linkid=2031032)
