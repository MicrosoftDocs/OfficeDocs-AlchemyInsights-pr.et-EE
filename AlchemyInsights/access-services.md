# <a name="access-services"></a>Juurdepääs teenustele

Teenuse Access Services funktsiooni SharePoint Online pensionisumma. Uue Accessi-põhise veebi rakenduste ja web andmebaase enam loomist ja olemasolevate veebirakenduste ja veebiandmebaasid tuleb sulgeda aprill, 2018. See muudatus ei mõjuta Desktop Accessi andmebaasi (.accdb). Soovitame Microsofti PowerApps juurdepääsu asemel veebi rakenduste ja web andmebaasid. 
  
[Siin on teavet Microsofti PowerApps](https://powerapps.microsoft.com/)
