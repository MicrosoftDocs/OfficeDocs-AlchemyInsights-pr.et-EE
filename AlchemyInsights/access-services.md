# <a name="access-services"></a>Juurdepääs teenustele

SharePoint Online ' i juurdepääsuteenuste funktsioon on pensionil. Uusi Accessi-põhiseid veebirakendusi ja veebiandmebaase ei saa enam luua ning olemasolevad veebirakendused ja veebiandmebaasid sulgub aprillis, 2018. See muudatus ei mõjuta juurdepääsu töölaua andmebaasid (. accdb). Soovitame Microsoft Powerappsi alternatiivina juurdepääsu veebirakenduste ja veebiandmebaasidele. 
  
[Teave Microsoft Powerappsi kohta](https://powerapps.microsoft.com/)
