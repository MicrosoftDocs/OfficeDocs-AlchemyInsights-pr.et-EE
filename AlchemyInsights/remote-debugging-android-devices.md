# <a name="use-microsoft-edge-to-remotely-debug-an-android-device-from-a-development-computer"></a>Microsoft Edge ' i kasutamine arvuti kaudu Androidi seadme silumiseks

Oma arvutist Androidi seadmes oleva sisu otse silumiseks tehke järgmist.

1. Häälestage Androidi seade serveri silumiseks ja Avastage see arengu arvuti kaudu.
2. Uurige ja siluda arvutis olevat reaalajas sisu Androidi seadmes.
3. Screencast Androidi seadmest reaalajas sisu DevTools eksemplarile.
