# <a name="add-or-remove-an-email-address-for-a-user"></a>Kasutaja meiliaadressi lisamine või eemaldamine

Alustamiseks [järgige neid juhiseid](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) Microsoft 365 halduskeskus.

 **Märkus**: kasutaja jaoks võib olla rohkem kui üks meiliaadress, kuid täiendavaid  *e-posti pseudonüüme*  saab kasutada ainult meilisõnumite vastuvõtmiseks. Kui teil on vaja saata pseudonüümiga meilisõnumeid, looge [ühiskasutuses postkast](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). Nii saad pseudonüümi meilisõnumite saatmiseks ja vastuvõtmiseks.
