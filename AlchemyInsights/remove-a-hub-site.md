# <a name="remove-a-hub-site"></a>Jaoturi saidi eemaldamine

Jaoturi saidi registreerimise tühistamiseks uues SharePointi administreerimiskeskuses tehke järgmist.
  
1. Klõpsake uues SharePointi administreerimiskeskuses jaotises **saidid**nuppu **aktiivsed saidid**.

2. Valige sait, klõpsake nuppu **jaoturi sait**ja seejärel klõpsake käsku Tühista **registreerimine jaoturi saidina**.

3. Klõpsake nuppu **OK**.

[Lisateave jaoturi saitide kohta](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
