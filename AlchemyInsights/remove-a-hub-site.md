# <a name="remove-a-hub-site"></a>Jaoturi saidi eemaldamine

Uue SharePointi administreerimiskeskuse jaoturi saidi registreerimise registreerimiseks toimige järgmiselt.
  
1. Klõpsake uue SharePointi administreerimiskeskus jaotises **saidid**, **aktiivsed saidid**.

2. Valige sait, klõpsake **jaoturi saidja**klõpsake **unregister kui jaoturi sait**.

3. Klõpsake nuppu **OK**.

[Lisateave jaoturi saitide kohta](https://support.office.com/article/what-is-a-sharepoint-hub-site-fe26ae84-14b7-45b6-a6d1-948b3966427f)
