# <a name="supported-version-of-the-sharepoint-migration-tool"></a>SharePointi siirdamise tööriista toetatud versioon



Praegu SharePointi migreerimise tööriist toetab ainult migreerimist SharePoint 2013. Versioonid 2010 ja 2016 ei toetata sel ajal.
  
Lisateabe saamiseks vaadake [Sissejuhatus SharePointi siirdamise tööriista](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
