# <a name="supported-version-of-the-sharepoint-migration-tool"></a>SharePointi migreerimise tööriista toetatud versioon



Praegu toetab SharePointi migreerimise tööriist ainult SharePointi 2013 migreerimist. Praegu ei toetata versioone 2010 ja 2016.
  
Lisateavet leiate teemast [SharePointi migreerimise tööriista tutvustus](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
