# <a name="deploy-microsoft-365-apps"></a>Microsoft 365 rakenduste juurutamine

Microsoft 365 rakenduste rakendamiseks mitmele lõppkasutajale Office ' i juurutamise tööriista kaudu, Microsoft Endpoint Configuration Manageri osana operatsioonisüsteemi ketta pildina või ühiskasutatavas arvutis, kus töötab Windows Server koos kaugtöölaua teenuste (RDS) või terminaliteenustega, kaaluge rakenduse **microsoft 365 Enterprise Deployment Advisor**juhiseid samm-sammult.
  
Alustamiseks veenduge, et olete sisse logitud rentniku administraatorina või üldadministraator, ja seejärel klõpsake siin: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
