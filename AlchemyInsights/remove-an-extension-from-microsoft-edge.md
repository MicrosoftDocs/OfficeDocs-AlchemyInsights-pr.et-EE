# <a name="remove-an-extension-from-microsoft-edge"></a>Laienduse eemaldamine Microsoft Edge'ist

Microsoft Edge'ist laiendi eemaldamiseks:

1. Avage Microsoft Edge.
2. Valige **... > Laiendid**.
3. Otsige **lehel Installitud laiendid** üles laiend, mille soovite eemaldada, ja seejärel **valige selle** kõrval Käsk Eemalda.
4. Klõpsake kinnituse dialoogiboksis nuppu **Eemalda**.
