# <a name="remove-an-extension-from-microsoft-edge"></a>Laiendi eemaldamine Microsoft Edge ' ist

Microsoft Edge ' i laiendi eemaldamiseks tehke järgmist.

1. Microsoft Edge ' i avamine
2. Valige **> laiendid**....
3. Leidke lehel **installitud laiendused** laiendus, mille soovite eemaldada, ja seejärel klõpsake nuppu **Eemalda** selle kõrval.
4. Klõpsake dialoogiboksis kinnitus nuppu **Eemalda**.
