# <a name="use-prt-based-sso-in-microsoft-edge"></a>PRT-põhise SSO kasutamine Microsoft Edge'is

Esmane [värskendustõend (PRT)](https://go.microsoft.com/fwlink/?linkid=2133632) on Azure Active Directory võti, mida kasutatakse autentimiseks Windows 10, iOS-i ja Androidi seadmetes. PRT võimaldab SSO-d kõigis nendes seadmetes kasutatavates rakendustes.

Microsoft Edge'is on PRT-põhise SSO tugi omane ja ei nõua täiendavaid laiendeid. Opsüsteemis Windows 10 RS3 ja eespool saavad sisse logitud kasutajad PRT-ga SSO-d kasutada prt-põhise SSO-ga veebisaitidel.
