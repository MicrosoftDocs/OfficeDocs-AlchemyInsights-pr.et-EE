# <a name="add-a-guest-to-microsoft-teams"></a>Külalise lisamine Microsoft Teamsi

Mõnikord soovite teha koostööd inimestega, kes pole teie meeskonna organisatsioonis. Sageli teete seda meili teel, kuid kui soovite nendega Teamsis koostööd teha, saate need lisada külalistena.

Lisateavet leiate teemast Külaliste lisamine töörühma [Teamsis.](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop)

**Märkus.** Teamsis külalise lisamiseks peate olema meeskonna omanik (vt ka teemat [Külalisõiguste määramine).](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)
