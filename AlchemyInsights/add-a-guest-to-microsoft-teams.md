# <a name="add-a-guest-to-microsoft-teams"></a>Microsoft Teamsi külalisena lisamine

Mõnikord soovite teha koostööd inimestega, kes pole teie meeskonna asutuses. Sageli saate seda teha meili teel, kuid kui soovite meeskonnas nendega koostööd teha, saate need lisada külaliste jaoks.

Lisateavet leiate teemast [külaliste lisamine meeskonda](https://support.office.com/article/add-guests-to-a-team-in-teams-fccb4fa6-f864-4508-bdde-256e7384a14f#ID0EAABAAA=Desktop).

**Märkus:** Teamsi meeskonna lisamiseks peate olema meeskonna omanik (vt ka teemast [Guest permissions](https://support.office.com/article/set-guest-permissions-for-channels-in-teams-4756c468-2746-4bfd-a582-736d55fcc169)).
