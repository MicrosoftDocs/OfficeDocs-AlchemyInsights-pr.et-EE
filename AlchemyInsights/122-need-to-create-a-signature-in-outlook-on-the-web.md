# <a name="need-to-create-an-email-signature"></a>On vaja luua meilisõnumi allkirja?

Kui teil on Outlook konto või Outlooki web (Office 365 tööl või koolis) konto, saate luua allkirja ja automaatselt lisada kõigile Väljaminevatele sõnumitele või valikuliselt teatud Väljaminevatele sõnumitele lisamine. Teadmiseks, et e-posti allkiri on Outlook ja Outlook Web. Teie e-posti allkirja loomiseks vt [Loo ja signatuuri lisamine sõnumitele Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) või [loomine ja e-posti allkiri lisada Outlook Web](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
