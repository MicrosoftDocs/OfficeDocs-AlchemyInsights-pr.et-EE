# <a name="troubleshoot-administrator-issues"></a>Tõrkeotsing administraatori probleemidele

Administraatoriga seotud probleemide tõrkeotsingu sooritamiseks tehke järgmist.

1. Oma kataloogis globaalsete administraatorite kuvamiseks avage leht **rollid ja administraatorid** .
2. Otsige lehelt **rollid ja administraatorid** rollide loendist **globaalne administraatori** roll. Avage **üldadministraatori** leht, et näha kõiki oma kataloogis olevaid globaalseid administraatoreid.
3. [Administraatori rollide kuvamine rakenduses AZURE ad](https://docs.microsoft.com/azure/active-directory/roles/manage-roles-portal): Nüüd saate vaadata ja hallata kõiki **administraatori rolli** liikmeid Azure AD portaalis.
4. Saate teada, kuidas Azure [ad rollide ja Azure ' i RBAC rollide erinevus](https://docs.microsoft.com/azure/role-based-access-control/rbac-and-directory-admin-roles): kui teil on uus Azure ' i jaoks, võib teil olla vähe väljakutseid Azure ' i erinevate rollide mõistmiseks. Selles artiklis selgitatakse erinevaid rolle ja kui soovite kasutada kõiki neid rolle.
5. [Uue Azure Active Directory loomine](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-access-create-new-tenant): selles kiirjuhend saate teada, kuidas pääseda juurde Azure ' i portaalile ja Azure ' i reklaamile, ning saate teada, kuidas luua oma asutuse jaoks tavaline rentnik.
6. [Azure ' i tellimuse administraatorite lisamine või muutmine](https://docs.microsoft.com/azure/cost-management-billing/manage/add-change-subscription-administrator): selles artiklis kirjeldatakse, kuidas lisada või muuta kasutaja administraatori rolli, kasutades Azure ' i RBAC tellimuse ulatuses.
