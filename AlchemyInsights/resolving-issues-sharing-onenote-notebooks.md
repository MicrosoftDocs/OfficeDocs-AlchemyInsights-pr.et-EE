# <a name="resolving-issues-sharing-onenote-notebooks"></a>OneNote'i märkmike ühiskasutusega seotud probleemide lahendamine

Nupp Anna ühiskasutusse asub OneNote'i paremas ülanurgas.

- Märkmiku ühiskasutusse andmiseks sisestage üks meiliaadress.
- Klõpsake nuppu **Anna ühiskasutusse**.

Märkmiku uuesti ühiskasutusse andmiseks sulgege märkmik ja avage see uuesti OneNote'is.
