# <a name="getting-an-exchange-hybrid-key"></a>Saada hübriid vahetusvõtme

Ühenduse teie asutusesisese organisatsiooni Exchange Online'i, Office 365, peate ühe Exchange Serveri Asutusesisese organisatsiooni seadistamiseks. See server saab luua hübriidjuurutuse, mis loob ühenduse teie asutusesisese Exchange'i organisatsiooni Exchange Online'i. Saada võti hübriid see server, järgige [Exchange hübriid toote jaotuse wizard](https://aka.ms/hybridkey)kell.
