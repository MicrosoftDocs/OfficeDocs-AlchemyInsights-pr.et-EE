# <a name="getting-an-exchange-hybrid-key"></a>Exchange ' i hübriidvõtme saamine

Asutusesisese organisatsiooni Exchange Online ' i ühendamiseks Microsoft 365, peate seadistama vähemalt üks Exchange ' i serveri Asutusesisese organisatsiooni. Seda serverit kasutatakse hübriid juurutamine, mis ühendab asutusesisese Exchange ' i organisatsiooni Exchange Online luua. Selle serveri hübriidvõtme saamiseks järgige [Exchange ' i hübriidtoote võtme levitamise viisardi](https://aka.ms/hybridkey)juhiseid.
