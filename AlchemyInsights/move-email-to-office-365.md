# <a name="move-email-to-microsoft-365"></a>E-posti teisaldamine Microsoft 365

Teil on mitu võimalust migreerimine e-posti Microsoft 365. [Microsoft 365 mail rände nõustaja](https://aka.ms/alchemyinsight-mailmigrationadvisor) aitab teil valida parim oma keskkonna ja juhendab teid läbi e-posti migreerimise.
  
Kui soovite lugeda oma valikute kohta ilma migreerimise käivitamata, siis vaadake [Otsustage rändetee](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Palun kontrollige ka [võtta vastu Microsoft 365 koos FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) lehekülg abiga abi rände, kui abikõlblikud.
