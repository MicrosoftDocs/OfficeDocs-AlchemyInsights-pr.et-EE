# <a name="assign-security-administration-permissions"></a>Turvalisuse halduse lubade määramine

Turvalisuse halduse lubade määramiseks tehke järgmist.

1. Valige [Office 365 turbe & täitmise keskuses](https://sip.protection.office.com/homepage) **õigused > Security Administrator**.
2. Klõpsake paanil **Security Administrator** **liikmete** kõrval nuppu **Redigeeri**.
3. Klõpsake paanil **Redigeeri liikmete valimine** käsku **Vali liikmed**.
4. Valige paanil **liikmete valimine** Cliok **+ Add**(kasutaja nimed) ja seejärel valige kasutajate nimed.
5. Valige **lisa > valmis > Salvesta**.
