# <a name="turn-an-add-in-off-for-outlook-for-windows"></a>Lisandmooduli väljalülitamine rakenduses Outlook for Windows

Kui te ei soovi rakenduses Outlook 2013 või Outlook 2016 Windowsi jaoks lisandmoodulit kasutada, saate selle keelata, et see ei kuvataks teie sõnumites.  

1. Klõpsake Outlookis nuppu **pilt > Lisandmoodulite haldamine**.
2. Klõpsake nuppu  **Manage Add-ins**
3. Näpunäide: See avab Outlooki veebirakenduses.
4. Tühjendage jaotises Lisandmoodulite haldamine veerus sisselülitatud säte selle **lisandmooduli ruut**  , mille soovite välja lülitada.
5. Lisandmoodulite sisselülitamine ja väljalülitamine

Näpunäide: lisandmooduli uuesti sisselülitamiseks märkige sama ruut.
