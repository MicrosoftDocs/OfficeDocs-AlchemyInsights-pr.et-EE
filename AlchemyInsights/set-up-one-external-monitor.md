# <a name="set-up-one-monitor"></a>Ühe kuvari häälestamine

**ARVUTI projekti sätete kontrollimine**

1. Avage oma Projecti sätted, vajutades samal ajal järgmisi klahve: Windowsi klahv (Windowsi logoga nupp Start) + P.
2. Sõltuvalt teie eelistustest valige ainult **duplikaat**, **pikenda**või **teine kuva**.

**Märkus:** Kui teie ARVUTIl on ainult üks kuvatav Port (nt HDMI või VGA), peate mitme kuvari häälestamiseks kasutama dokkimisjaama või USB-adapterit.

Kui saate häälestada ühe kuvari, kuid mitte mitu, ei pruugi teie arvuti mitme kuvariga ühilduda. Lisateavet leiate oma arvuti algsetest dokumentidest.
