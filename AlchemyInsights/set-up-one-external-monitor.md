# <a name="set-up-one-monitor"></a>Ühe kuvari häälestamine

**ARVUTI projekti seadete kontrollimine**

1. Avage oma projekti sätted, vajutades samal ajal järgmisi klahve: Windowsi klahv (Windowsi logo Start button) + P.
2. Valige **duplikaat**, **Laienda**või **teine ekraan**, sõltuvalt teie eelistustest.

**Märkus:** Kui teie ARVUTIL on ainult üks kuvasport (nt HDMI või VGA), vajate mitme kuvari seadistamiseks dokkimisjaama või USB-adapterit.

Samuti, kui saate häälestada ühe kuvari, kuid mitte mitu, ei pruugi teie arvuti olla ühilduv mitme kuvariga. Vaadake lisateavet oma arvuti originaaldokumentidest.
