# <a name="why-cant-i-edit-this-file"></a>Miks ma ei saa seda pilti redigeerida?

Kui dokument, tööleht või esitlus on redigeerimiseks lukustatud ja seda ei saa Windowsi töölauarakenduse kaudu redigeerida, võib see olla see, et keegi teine on faili redigeerinud või on fail välja möllitud.

Vaadake kiiresti mõnda põhjust, [miks te ei pruugi faili redigeerida](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
