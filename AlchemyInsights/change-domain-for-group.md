# <a name="change-the-domain-for-microsoft-365-group"></a>Microsoft 365 rühma domeeni muutmine

Kui soovite loomise ajal muuta Microsoft 365 rühma domeeni, järgige juhiseid jaotises Valige domeen, mida [kasutada Microsoft 365 rühmade loomisel.](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups)
  
Kui Microsoft 365 rühm on juba loodud, pole praegu võimalik domeeni muuta. Ainus võimalus on luua uus Microsoft 365 rühm ja vajaduse korral kustutada vana (exchange'i halduskeskuse kaudu saab lisada [täiendavaid meiliaadresse).](https://outlook.office365.com/ecp.aspx)
