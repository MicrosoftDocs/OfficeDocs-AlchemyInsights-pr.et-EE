# <a name="powerapps-and-microsoft-flow-plan-extension"></a>Powerappsi ja Microsoft Flow plaani laiend

Kui kavatsete taotleda plaani laiend **Microsoft Flow** või **powerapps** ja teie rentnik juba on aktiivsed kasutajad üks järgmistest funkttest, ei ole vaja esitada tugiteenuse taotlus.

- Loomine ja avaldamine kohandatud konnektorid PowerApps ja Microsoft Flow
- HTTP kohandatud toiminguid integreerida Flow väljaspool SharePointi ja OneDrive
- Asutusesiseste andmete integreerimine asutusesisese andmelüüsi kaudu

Nende funktsioonide aktiivsete kasutajatega kliendid saavad automaatse pikendamise kuni jaanuarini 31, 2020 või oma olemasoleva Microsoft 365 tellimuse kehtivusaja aegumiseni, olenevalt sellest, kumb on pikem.
