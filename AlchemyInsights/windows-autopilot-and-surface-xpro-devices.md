# <a name="windows-autopilot-and-surface-x-pro-devices"></a>Windows autopiloot ja Surface X Pro seadmed

Windows autopiloot on Windows 10 pilvepõhise juurutuse tehnoloogia. Saate kasutada rakendust Windows autopiloot, et kaugjuhtida ja konfigureerida seadmed otse karbist välja. Windows autopiloot registreeritud seadmed tuvastatakse Internetis esimesel käivitamisel ainulaadse seadme signatuuri abil, mida nimetatakse riistvara räsiks. Nad on konfigureeritud kasutama kaasaegseid halduse lahendusi (nt Azure Active Directory (Azure AD) ja Microsoft Intune ' i).

Seadmeid saab saata otse teie kasutajatele. See protsess kõrvaldab reimaging juurutamise ajal, mis võimaldab teil rakendada uusi, dünaamilisi seadmete halduse ja levitamise meetodeid.

Lisateavet leiate teemast [Windows autopiloot ja Surface](https://go.microsoft.com/fwlink/?linkid=2135712)' i seadmed.
