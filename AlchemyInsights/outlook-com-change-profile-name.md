# <a name="change-your-profile-name-in-outlookcom"></a>Outlook.com-i profiili nime muutmine

1. Avage [oma Microsofti konto teabe](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Valige **Muuda nimi**.
3. Sisestage eesnimi ja perekonnanimi ja valige **Salvesta**.

Palun lubage kuni 24 tundi teie profiilinime sisse muudatused.
