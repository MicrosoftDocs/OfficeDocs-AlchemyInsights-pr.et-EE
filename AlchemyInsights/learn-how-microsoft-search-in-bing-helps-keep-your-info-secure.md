# <a name="learn-how-microsoft-search-in-bing-helps-keep-your-information-secure"></a>Juhised selle kohta, kuidas Microsoft Search Bingis aitab teie teavet turvalisena hoida

Microsoft Searchi abil Bingis tehke järgmist.

- **Otsingutulemusi** saavad vaadata ainult sisselogitud kasutajad.
- Microsoft ei kasuta teie tööidentiteeti ega teie asutuse identiteeti sihtrühma reklaamimiseks.
- Teie otsingu administraator saab vaadata vaid otsingute arvu tüübi järgi (nt isiku või failitüübi alusel) ja ülemise otsingute koondatud loendit.
- Ainult teie töö **otsinguajalugu** kuvatakse.
- Saate otsida ainult teile ühiskasutusse antud tööfaile ja teavet või teil on õigus seda vaadata.
- Teie teave ei muutu kunagi kättesaadavaks teistele organisatsioonidele (sh Microsoftile) või Bingis teostatavatele avalikele otsingutele.

Lisateavet leiate teemast [Kuidas Microsoft Search Bingis aitab teie teavet turvalisena hoida](https://support.microsoft.com/office/how-microsoft-search-in-bing-helps-keep-your-info-secure-cbce46ae-bb1f-4d0e-86f1-5984f4589113)?.
