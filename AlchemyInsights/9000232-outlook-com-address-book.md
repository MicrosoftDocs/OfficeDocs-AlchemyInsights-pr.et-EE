# <a name="address-book-in-outlookcom"></a>Aadressiraamatu Outlook.com-is

Teie Outlook.com-i aadressiraamatu nimetatakse inimest.

Lehel inimesed, valige **inimesed** alumise nurga lehekülje või lehe ülanurgas rakenduse paigutusselektorit.

Õppida, kuidas [Loo, arvates ja kontaktandmete muutmine Outlook.com-is](https://support.office.com/article/5b909158-036e-4820-92f7-2a27f57b9f01).
