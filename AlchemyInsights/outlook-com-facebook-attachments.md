# <a name="unable-to-attach-or-upload-files-from-facebook"></a>Ei saa lisada või üles laadida faile Facebook

Tänu muutus Facebook tingimused, pärast oktoober 23, 2019, sa ei saa enam võimalik üles laadida manuseid otse Facebook, ja Facebook ei ilmu enam nimekirja oma pilve asukohad. 

Oma [salvestusruumi kontodel](https://go.microsoft.com/fwlink/?linkid=2111075)saate hallata oma teiste pilvekohtade loendit.

Vaadake lisateavet [salvestusruumi kontode](https://support.office.com/article/477cb7cc-5732-4c40-8f23-30472de8138a)kohta.
