# <a name="set-up-windows-virtual-desktop"></a>Windowsi virtuaalse töölaua häälestamine

Windows Virtual Desktop on terviklik lauaarvuti ja rakenduste virtualiseerimise teenus, mis töötab pilves. See on ainus Virtual Desktopi infrastruktuur (VDI), mis pakub lihtsustatud haldust, multi-session Windows 10, optimeerimine Microsoft 365 apps ja tugi Remote Desktop Services (RDS) keskkondades. Juurutage ja mastaabige Windowsi töölaudu ja rakendusi Azure ' i minutites ning Hankige sisseehitatud turbe-ja vastavuskontrolli funktsioonid.

[Windows Virtual Desktop setup guide](https://go.microsoft.com/fwlink/?linkid=2146236) pakub administraatoritele planeerimiseks ressursse ning eeltingimusi juurutuse, häälestamise juhiste ja lisaressursside jaoks.
