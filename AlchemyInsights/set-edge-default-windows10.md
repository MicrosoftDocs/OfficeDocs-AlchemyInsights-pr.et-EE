# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>Microsoft Edge'i seadis Windows 10 seadme vaikebrauseriks

Tehke järgmist.

1. Valige nupp **Start** ja seejärel sisestage **vaikerakendused.**

1. Valige otsingutulemites **Vaikerakendused**.

1. Klõpsake **jaotises Veebibrauser** nuppu **Microsoft Edge**.
