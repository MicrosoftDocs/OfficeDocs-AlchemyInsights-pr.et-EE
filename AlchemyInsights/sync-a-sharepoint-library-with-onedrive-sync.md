# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>SharePointi teegi sünkroonimine OneDrive ' i sünkroonimisel

Selleks, et teie SharePointi failid sünkroonida oma kohalikule kõvakettale, tehke järgmised sammud:

1. Liikuge dokumenditeeki, mida soovite sünkroonida oma kohalikule kõvakettale.
2. Valige menüüsuvanditest nupp Sünkrooni.
3. Sünkroonimise lõpuleviimiseks järgige samm-sammult.
