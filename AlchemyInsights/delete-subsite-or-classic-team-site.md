# <a name="delete-a-classic-sharepoint-team-site-or-subsite"></a>Klassikaline SharePointi meeskonnatöö veebisaidi või alamsaidi kustutada

1. Minge veebisaidile.
    
2. Klõpsake paremas ülanurgas **seadete** ikoonil. 
    
3. Klõpsake käsku **Saidisätted**. Jaotises **Saiditoimingud**käsku **Kustuta see sait**ja seejärel klõpsake käsku **Kustuta**.
