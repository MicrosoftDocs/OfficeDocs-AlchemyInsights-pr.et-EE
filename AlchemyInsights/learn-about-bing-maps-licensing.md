# <a name="learn-about-bing-maps-licensing"></a>Lisateave Bingi kaartide litsentsimise kohta

Bingi kaartide teenus pakub mitmeid paindlikke litsentsimise suvandeid, mis toetavad teie asutuse vajadusi ja tehnilisi nõudeid. Samuti saate tasuta tasemega alustada juba täna Bingi kaartidega hooneid.

Lisateavet leiate teemast [Bingi kaartide litsentsid](https://go.microsoft.com/fwlink/?linkid=2150203).
