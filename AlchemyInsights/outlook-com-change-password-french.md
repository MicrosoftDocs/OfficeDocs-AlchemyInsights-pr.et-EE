# <a name="changer-votre-mot-de-passe-dans-outlookcom"></a>Box votre mot de passe dans Outlook.com

Votre mot de passe Outlook.com est le même que votre mot de passe Compte Microsoft.

1. Accédez à [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) ja sélectionnez **muutja le mot de passe**.
2. Par mesure de sécurité, vous pouvez mõte invité à ka tellimuse kinnitajale votre identité à l'aide toetuse kood de sécurité.
3. Entrez votre mot de passe actuel, votre nouveau mot de passe, puis sélectionnez **Enregistrer**.

Si vous avez oublié votre mot de passe, accédez à la [lehe de réinitialisation de mot de passe de compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909), puis suivez les kutsub.
