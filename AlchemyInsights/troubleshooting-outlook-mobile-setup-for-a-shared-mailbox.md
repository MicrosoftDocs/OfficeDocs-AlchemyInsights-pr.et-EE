# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Tõrkeotsing Outlook Mobile setup ühiskasutusse antud postkasti

Te ei pääse ühiskasutusse antud postkasti Outlook rakendusest, mis on installitud mobiilsideseadme. Kas soovite seda funktsiooni? Las Outlook Engineering meeskond teab. [Hääletage siin](https://go.microsoft.com/fwlink/?linked=862116)!
  
Ühiskasutusega postkasti saate kasutada ainult mobiilseadmes, avades oma brauseri, logite sisse Office 365 ja seejärel minge Outlook veebis. Juhised leiate jaotisest [ühiskasutusega postkasti lisamine Outlookis veebis](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Outlookist veebis pääsete ühiskasutusse antud postkasti.
  
Lisateave: [Hankige rakendusesiseseid spikrit Outlooki iOS-i ja Androidi](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243) jaoks
