# <a name="change-a-users-name-and-email-address"></a>Muuta kasutaja nimi ja aadress

Kasutaja nimi ja e-posti aadressi muutmine on lihtne! Mine [Aktiivsed kasutajad](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users) ja valige kasutaja muuta oma andmeid. Samuti saate selle artikli üksikasjalikud juhised: 
  
[Office 365 nimi ja e-posti aadressi muutmine](https://support.office.com/article/Change-a-user-name-and-email-address-in-Office-365-fb5ac074-e203-4e1f-9843-b9d1a3e03297?wt.mc_id=change_email_AI.aspx)
  
 **Märkus**: kui kasutaja peamist meiliaadressi muuta oma vana e-posti aadress ikka kehtib (see toimub automaatselt) meili vastuvõtmiseks. Kui teie organisatsioonis on hübriid olekus, kõik e-posti ja kasutaja nime muutused tuleb teha asutusesisese Active Directory ja seejärel sünkroonitud pilve.
