# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>PowerAppsi või Microsoft Flow'i tugiteenusetaotluste esitamine

Kui olete **PowerAppsi või** **Microsoft Flow'i** klient, esitage oma pilet uue [Power Platformi halduskeskuse kaudu.](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819) Uus portaal pakub otsest teed toetatud toele, mis on optimeeritud PowerAppsi ja muude Power'i platvormi toodete jaoks.
