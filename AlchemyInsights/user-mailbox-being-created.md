# <a name="your-users-mailbox-is-being-created"></a>Teie kasutaja postkast on loodud

See võib kuluda kuni tund enne, kui Uus postkast on kasutamiseks valmis.
  
Kui kasutaja postkast on "ettevalmistamine" rohkem kui tund, võite proovida kustutada ja uuesti luua kasutaja või kasutada "Olgem helistada" võimalus rääkida tugiteenuse insener.
