# <a name="how-to-create-a-rule-in-outlook-for-mac"></a>Reegli loomine rakenduses Outlook for Mac

1. Reegliga saadetud meilisõnumite hoidmiseks kausta loomiseks valige root postkast, klõpsake menüü **korraldamine** nuppu **Uus kaust**, tippige kausta nimi ja vajutage sisestusklahvi (ENTER).

2. Klõpsake sõnumiloendis meilisõnumit, mille jaoks soovite o reegli luua.

3. Klõpsake menüüs **Avaleht** nuppu **reeglid**ja seejärel käsku **Loo reegel**.

4. Jaotises **uue sõnumi saabumisel** redigeerige kriteeriume, et need vastaksid teie vajadustele. 

Üksikasjalikke juhiseid kohandatud reeglite loomise kohta leiate teemast [reegli loomine rakenduses Outlook for Mac](https://aka.ms/AA1uy0v)
