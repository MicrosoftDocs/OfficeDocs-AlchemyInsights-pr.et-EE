# <a name="project-web-app-is-missing-from-the-sharepoint-site-collection"></a>Project Web App puudub SharePointi saidikogumi

Project Web App võib puududa SharePointi saidikogumi järgmistel põhjustel:

1. Organisatsioonidel on ainult Project Online Essentialsi litsents (ID). Administraatoritel on vaja lisada Project Online Professional ja/või Premium litsents. Sait tuleb välja kirjutuskaitstud kohas. Lisateavet leiate teemast [projektihalduse lahenduste võrdlemine](https://products.office.com/project/compare-microsoft-project-management-software?tab=1).
2. Rentnik on tekkinud ettevalmistamise probleemi, mille Project Web Appi saidi ettevalmistamine on hilinenud. Kui möödunud on üle 24 tunni ja teil on endiselt probleem, Palun logige tugiteenuse pilet. Me mõistame, et 24 tundi võib tunduda pikk aeg. Paljudel juhtudel oleme juba lahenduse kallal töötanud. Palun andke meile vähemalt 24 tundi, et lahendus lõpule viia.
