# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>Millal muutub profiili sünkroonimine SharePointi kasutajaprofiili rakendusega?

Profiili pildi sünkroonimine SharePoint Online ' is nõuab kasutaja tegevust, et alustada oma foto sünkroonimist. SharePoint Online ' i fotosid ei sünkroonita automaatselt. SharePoint Online sünkroonib ainult fotosid Exchange Online ' ist, mitte Azure Active Directory ' st.

Kui kasutaja pole kunagi sooritanud toimingut, mis viipaks mõnele sünkroonimisele või kui nad on oma profiili pilti värskendanud ja uut versiooni SharePoint Online ' is ei kuvata, võib kasutajal tekkida vajadus see sünkroonimiseks paluda. Seda saab teha järgmiselt.

1. Avage veebibrauser privaatses või inkognito seansis ja avage [Office.com](https://www.office.com/).

2. Logige sisse oma töö-või kooli kontoga ja klõpsake OneDrive ' i või SharePointi paani.

3. [Lisage oma profiili foto Microsoft 365](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

[Teave profiili pildi sünkroonimise kohta rakenduses Microsoft 365](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)
