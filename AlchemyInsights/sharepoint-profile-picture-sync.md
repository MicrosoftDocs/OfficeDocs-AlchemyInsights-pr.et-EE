# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>Millal muudab profiil SharePointi kasutajaprofiili rakenduse sünkroonimist?

Profiili pildi sünkroonimine SharePoint Online ' i nõuab kasutaja toiming algatada sünkroonimise oma foto. Ei ole Automaatne sünkroonimine fotosid SharePoint Online. SharePoint Online ainult sünkroonib fotosid Exchange Online ' i, mitte Azure Active Directory.

Kui kasutaja pole kunagi teinud toimingut, mis küsiks sünkroonimise või nad on värskendatud oma profiilipilti ja uus ei kuvata SharePoint Online ' i, kasutaja võib olla vaja paluda seda sünkroonida. Nii saavad nad teha järgmist:

1. Avage veebibrauser in-Private või inkognito seansi ja [Office.com](https://www.office.com/).

2. Logige sisse oma töö või kooli kontoga ja klõpsake OneDrive ' i või SharePointi paani.

3. [Lisage oma profiilifoto Microsoft 365](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

[Teave profiilipildi sünkroonimise kohta rakenduses Microsoft 365](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)
