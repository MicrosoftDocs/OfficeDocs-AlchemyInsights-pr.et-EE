# <a name="create-a-team-in-microsoft-teams"></a>Microsoft Teamsi töörühma loomine

Microsoft teamsi töörühma loomiseks tehke järgmist.

1. Klõpsake meeskondade vasakus servas nuppu **töörühmad**.
2. Klõpsake rühmade loendi allservas nuppu **Liitu** või **Loo rühm**.
3. Klõpsake nuppu **Loo uus rühm**.

Kui teil on töörühma loomisega probleeme, võib olla rakendatud poliitika. Lisateavet leiate teemast [Microsoft 365 rühmade loomise võimalus](https://docs.microsoft.com/microsoft-365/solutions/manage-creation-of-groups).
