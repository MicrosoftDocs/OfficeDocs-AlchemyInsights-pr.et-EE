# <a name="multiple-active-sessions-to-the-same-mailbox"></a>Mitu aktiivset seanssi samale postkastile

Exchange ' i ressursside kasutuse juhtimiseks on postkastis "eelarve".

Üle-eelarve erandi saab käivitada, kuid see ei tohi olla piiratud järgmiste asjaoludega.

- Sama Outlook Web Appi seansi ajal avatakse mõne brauseri vahekaardid.

- Mõned aktiivsed Outlook Web Appi seansid samale postkastile.

- Mõned teised klientrakendused (Outlook, Outlook Mobile, kolmanda osapoole klientrakendus) pääsevad postkastile juurde samal ajal.

- Pikad töötavad toimingud (nt päringute täitmine) tehakse mõnest muust aktiivsest postkasti seansist.
