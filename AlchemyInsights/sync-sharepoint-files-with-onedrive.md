# <a name="troubleshoot-open-with-explorer-issues-in-sharepoint-online"></a>Käsu Ava Exploreris probleemide tõrkeotsing SharePoint Online’is

Käsk Ava Exploreris avab Windows Exploreri kohaliku eksemplari, mis kuvab SharePointi saiti majutava serveri kausta struktuuri. Seetõttu soovitame [sünkroonida SharePointi failid uue OneDrive’i sünkroonimisrakendusega](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)</a>, mis pakub [nõudefaile](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e), kuna see võimaldab failidele kohalikku juurdepääsu ja pakub parimat jõudlust.


Kui otsustasite uue sünkroonimisrakenduse asemel kasutada Exploreri vaadet, siis veenduge, et järgiksite järgmistes artiklites toodud juhendeid ja häid tavasid:

- [SharePoint Online’i probleemide tõrkeotsinguks käsu Ava Exploreris kasutamine](https://docs.microsoft.com/sharepoint/support/lists-and-libraries/troubleshoot-issues-using-open-with-explorer)

- [Käsu Ava Exploreris abil teegifailide kopeerimine või teisaldamine](https://support.office.com/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2)

> [!Note]  
> Uues teegi versioonis nuppu **Ava Exploreris** ei ole. Klõpsake üleval paremal rippmenüüd **Vaade** (rippmenüü nimi muutub olenevalt teie praegusest vaatest) ja seejärel valige **Vaade File Exploreris**.
>
 >SharePointi käsk Ava Exploreris kasutab ActiveX-juhtelemente, mistõttu seda toetatakse ainult Internet Exploreri versioonis 10 või 11. Käsk Ava Exploreris ei tööta Windowsis Microsoft Edge’i, Google Chrome’i, Mozilla Firefoxi ega Maci platvormiga. Seepärast võib suvand Exploreri vaade olla kuvatud tuhmina.
>
> - [Miks pole SharePointi lindi nupud kättesaadavad või on tuhmid](https://support.office.com/article/Why-SharePoint-ribbon-buttons-are-unavailable-48b0939a-2efb-4e79-b5e8-b2c4cb5d04ca)?
