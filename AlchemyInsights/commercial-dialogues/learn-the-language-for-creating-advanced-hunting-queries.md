# <a name="learn-the-language-for-creating-advanced-hunting-queries"></a>Täpsemate jahipidamise päringute loomise keel

Täpsem jaht põhineb [Kusto päringu keelel](https://go.microsoft.com/fwlink/?linkid=2144620). Kusto süntaksi ja tehtemärkide abil saate koostada päringuid, mis leiavad täpsemaks jahiks spetsiaalselt struktureeritud [skeemi](https://go.microsoft.com/fwlink/?linkid=2144621) teabe.

Lisateavet leiate teemast [Täpsem jahi päringu keel](https://go.microsoft.com/fwlink/?linkid=2144518).
