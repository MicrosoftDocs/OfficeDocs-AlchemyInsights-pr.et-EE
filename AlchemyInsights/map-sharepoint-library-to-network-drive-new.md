# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Kaarti võrgudraivile SharePointi teegis

Kaardistamine Raamatukogu network Drive on ajutine ja toetatud ainult Internet Exploreri kaudu. Aeg-ajalt tuleb avage SharePointi sait Internet Exploreris ja valige "Hinnast sisse" vältimiseks seanss lõpeb. Selle asemel [SharePointi failid koos uue OneDrive'i sünkroonimisrakenduse](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> pakub [Faile tellitava](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Juurdepääs oma OneDrive'is olevaid faile ilma kohalikku talletusruumi.

Kui otsustate draivi asemel[Uus OneDrive'i sünkroonimisrakenduse](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), tagada juhiste artikli alla.
