# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Kaarti võrgudraivile SharePointi teegis

Kaardistamine Raamatukogu network Drive on ajutine ja toetatud ainult Internet Exploreri kaudu. Aeg-ajalt tuleb avada SharePointi saidi Internet Explorer ja valige **sisselogituks** kuni seansi takistada. Selle asemel [SharePointi failid koos uue OneDrive'i sünkroonimisrakenduse](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> pakub [Faile tellitava](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Juurdepääs oma OneDrive'is olevaid faile ilma kohalikku talletusruumi.

Kui otsustate draivi asemel [Uus OneDrive'i sünkroonimisrakenduse](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), tagada juhiste artikli alla. 


**Vastendatud konfigureerimine ja tõrkeotsing**


Vaata [konfigureerida ja tõrkeotsing vastendatud](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

Märkus: Kui kasutate Internet Explorer 10 Windows 8 või Windows 7 ja saate draivi vastendamine **juurdepääs keelatud** või **tee on ligipääsmatu** , installige [See käigultparandus](https://support.microsoft.com/help/2846960) probleemi lahendamiseks.
