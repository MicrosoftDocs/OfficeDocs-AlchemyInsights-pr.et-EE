# <a name="security-defaults"></a>Turbe vaikesätted

Lisateavet turbe vaikesätete kohta leiate teemast [mis on turbe vaikesätted?](https://docs.microsoft.com/azure/active-directory/conditional-access/concept-conditional-access-security-defaults).

Azure Active Directory (Azure AD) turbe vaikesätted hõlbustavad turvalisust ja aitavad kaitsta teie ettevõtet. Turbe vaikeväärtused sisaldavad levinumate rünnakute eelkonfigureeritud turvasätteid.

Microsoft teeb kõigile kättesaadavaks turvalisuse vaikesätted. Eesmärk on tagada, et kõikidel organisatsioonidel oleks tavaline turvatase ilma lisatasuta. Lülitate sisse turbe vaikesätted Azure ' i portaalis.
