# <a name="learn-about-external-calendar-sharing-options"></a>Lisateave väliste kalendrite ühiskasutuse valikute kohta

Kui teil on vaja ajastada kohtumisi inimestega väljaspool oma organisatsiooni, Microsoft 365 võimaldab teil teha kalendrid kättesaadavaks "välised kasutajad" need, kes peavad nägema vaba/hõivatud aeg, kuid ei ole Kasutajakontod teie Microsoft 365 keskkonnas. Lisateavet leiate jaotisest [kalendrite ühiskasutus väliste kasutajatega](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
