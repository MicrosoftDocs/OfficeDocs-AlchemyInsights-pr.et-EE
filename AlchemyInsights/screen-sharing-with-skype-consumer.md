# <a name="screen-sharing-with-skype-consumer-users"></a>Ekraani jagamine Skype ' i tarbijate kasutajatega

Üldjuhised selle kohta, kuidas oma programmi ühiskasutusse anda, teavet ühiskasutuse piirangute ja ressursside tõrkeotsingu kohta leiate teemast [ekraani jagamine Skype ' i](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc)ärirakenduses.  

Praegu ei saa Skype ' i ärirakenduse kasutajad ekraani jagada Skype ' i tarbijate kasutajatega. Palun [klõpsa siin](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) , et anda tagasisidet selle kohta, kuidas võimaldada ekraani jagamine tarbija Skype ' i kasutajatega.
