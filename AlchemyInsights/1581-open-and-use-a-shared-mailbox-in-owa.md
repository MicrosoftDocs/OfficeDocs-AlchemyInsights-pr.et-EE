# <a name="open-and-use-a-shared-mailbox-in-outlook-on-the-web"></a>Avada ja kasutada ühiskasutusse antud postkasti Outlook Web

Ühiskasutuses postkasti, et teil on juurdepääs ei kuvata automaatselt Outlook Web (varasema nimega Outlook Web App).

Saate kuvada ühiskasutatava ja esmase postkasti koos või eraldi brauseriaknas avada ühiskasutatava. Uued teated ühiskasutatava saamiseks peate avada ühiskasutatava eraldi brauseriaknas. Lisateabe saamiseks vt [avatud ja kasutamise ühiskasutusse antud postkasti Outlook Web Appis](https://support.office.com/article/BC127866-42BE-4DE7-92AE-1EF2F787FD5C).
