# <a name="create-or-migrate-update-rings"></a>Update ' i helinate loomine või migreerimine

Kui olete konfigureerinud Windows 10 värskenduse sätted Intune ' i klassikalises portaalis ja soovite migreerida sätted Azure ' i portaalis Intune ' i, järgige neid juhiseid.

1.  Avage Azure ' i portaal ja valige **kõik teenused**.
2.  Tippige väljale **filter** tekst **Intune**ja valige **Microsoft Intune**.
3.  Valige **Software**Updates   >   **Windows 10 Update heliseb**   >   **loomine**.
4.  Sisestage nimi ja kirjeldus ning klõpsake nuppu **Konfigureeri**.
5.  Konfigureerige oma ettevõtte tarkvara värskendamise sätted.
6.  Valige **OK**  >  **Create Update ring**  >  **Create**.
