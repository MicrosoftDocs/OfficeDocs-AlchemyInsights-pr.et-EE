# <a name="powerapps-and-microsoft-flow-plan-extension"></a>PowerApps ja Microsoft Flow kava pikendamine

Kui te kavatsete taotleda kava laiendamist **Microsoft voolu** või **PowerApps** teie rentniku juba kasutajaid on üks järgmistest funktsioonidest, ei ole vaja esitada toetuse taotlus.

- Loomine ja avaldamine PowerApps ja Microsoft Flow kohandatud konnektorid
- HTTP kohandatud toiminguid voolu väljaspool SharePointis või OneDrive integreeritud
- Kohapeal andmeid kohapeal andmete lüüsi kaudu

Hotelli kasutajaid on need funktsioonid saavad automaatne pikendamine kuni 31 jaanuar 2020 või olemasoleva Office 365 tellimuse ametiaja lõppemisel, kumb periood on pikem.
