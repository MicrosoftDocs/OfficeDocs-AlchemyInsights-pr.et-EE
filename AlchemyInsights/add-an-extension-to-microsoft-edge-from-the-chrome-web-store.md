# <a name="add-an-extension-to-microsoft-edge-from-the-chrome-web-store"></a>Microsoft Edge ' i laienduse lisamine Chrome ' i veebipoest

Microsoft Edge ' i laienduse lisamiseks Chrome ' i veebipoest tehke järgmist.

1. Avage Chrome ' i veebipood.
2. Valige lehe ülaosas asuval sildil **Luba laiendid muudest poodidest**.
3. Valige laiend, mille soovite lisada, ja seejärel klõpsake nuppu **Lisa Chrome ' i**.
4. Vaadake kinnituse dialoogiboksis hoolikalt läbi muudatused, mida laiend võib muuta, ja klõpsake siis nuppu **Lisa laiend**.
Lõplik dialoog kinnitab, et laiend on lisatud.
