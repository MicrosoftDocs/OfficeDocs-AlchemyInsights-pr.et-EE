# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Rakenduse Outlook Mobile rakenduse tugiteenuste hankimine

1. Avage Outlook Mobile ' i rakenduses **menüü**.

2. Valige Kuva ** &amp; tagasiside** Kuva avamiseks küsimärk.

3. Puudutage nuppu **Võta ühendust kasutajatoega** ja selgitage oma probleemi.

4. Puudutage nuppu **saada** ja te saate tugiteenuste agendiga ühendust, kui tugiteenuste agent teie poole pöördub.

Üksikasjalikud juhised leiate teemast [Outlooki iOS-i ja Androidi rakenduse spikri hankimine](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support).
