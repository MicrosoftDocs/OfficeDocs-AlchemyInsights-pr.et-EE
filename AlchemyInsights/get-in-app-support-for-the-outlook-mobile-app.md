# <a name="get-in-app-support-for-the-outlook-mobile-app"></a>Rakendusetoe saamine Outlooki mobiilirakenduse jaoks

1. Avage Outlooki mobiilirakenduses **menüü**.

2. Spikri tagasiside kuva avamiseks **valige &amp; küsimärk.**

3. Puudutage **valikut Võtke ühendust** toega ja selgitage oma probleemi.

4. **Puudutage** nuppu Saada ja teile saadetakse rakenduse sees teatis, kui tugiagent teie poole kontakte võtab.

Üksikasjalikud juhised leiate teemast [Rakenduse spikri saamine iOS-i ja Androidi jaoks mõeldud Outlooki jaoks.](https://support.office.com/article/218a22d1-9fa5-4889-b689-de1c63493243.aspx#ID0EAABAAA=Contact_Support)
