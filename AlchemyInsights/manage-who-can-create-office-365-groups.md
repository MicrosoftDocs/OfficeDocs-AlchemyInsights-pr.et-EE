# <a name="manage-who-can-create-microsoft-365-groups"></a>Halda, kes saavad luua Microsoft 365 rühmad

Vaikimisi võib igaüks teie organisatsioonis luua O365 rühmi.
  
Kui soovite seda muuta, saate luua turberühma PowerShelli, mis piirab rühma loomine kasutajate alamhulga.
  
Järgige samm-sammult juhiseid siin: [Halda, kes saavad luua Microsoft 365 rühmad](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
