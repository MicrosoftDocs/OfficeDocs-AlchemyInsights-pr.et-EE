# <a name="delete-a-communication-site"></a>Side saidi kustutamine

1. Minge veebisaidile.
    
2. Klõpsake paremas ülanurgas seadete ikoonil.
    
3. Klõpsake käsku **Saidisätted**. Jaotises **Saidi teave**nuppu **Kustuta see sait**ja seejärel klõpsake käsku **Kustuta**.
