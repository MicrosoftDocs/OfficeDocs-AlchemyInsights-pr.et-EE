# <a name="create-a-site"></a>Saidi loomine

- Avage [Microsoft 365 halduskeskus lehel saidid](https://portal.office.com/adminportal/home#/SitesList)ja valige **Lisa sait**. 
    
- Klõpsake uues SharePointi administreerimiskeskuses lehel saidid nuppu **Loo**. 
    
**Märkus:** Te ei saa luua uut saiti, millel on olemasoleva saidiga sama URL. 
  
Lisateavet saidi loomise kohta leiate teemast [saidi loomine](https://go.microsoft.com/fwlink/?linkid=866295).
  
Lisateavet selle kohta, kuidas kasutajad saavad oma saite luua, leiate teemast [saidi loomise haldamine SharePoint Online ' is](https://go.microsoft.com/fwlink/?linkid=866296).
