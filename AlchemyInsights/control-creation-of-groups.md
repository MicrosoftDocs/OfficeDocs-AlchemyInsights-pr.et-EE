# <a name="control-creation-of-microsoft-365-groups"></a>Microsoft 365 rühmade loomise reguleerimine

Vaikimisi saavad kõik ettevõtte kasutajad luua rühmi. Vajadusel saate seda muuta ja määrata õiguse luua rühmi oma ettevõttes olevatele inimestele. Pange tähele, et see mõjutab rühmade loomist mis tahes Microsoft 365 teenusest (nt Outlook, Teams, Planner jne).
  
Juhised: [saate hallata, kes saavad Microsoft 365 rühmi luua](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
