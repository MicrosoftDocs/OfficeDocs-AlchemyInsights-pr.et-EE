# <a name="permissions-in-the-sharepoint-modern-experience"></a>SharePointi moodsa kogemuse õigus

Tavaliselt hallatakse SharePointi õiguseid saidi (omanike, liikmete, Külastajate jne) saitide kogumi kaudu. Microsoft 365 rakenduses SharePoint kehtib see teatud tüüpi saitide puhul, kuid saadaval on veel võimalusi.  

Lisateavet leiate SharePointi moodsast veebirakendusest, [ühiskasutusest ja õigustest](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions).
