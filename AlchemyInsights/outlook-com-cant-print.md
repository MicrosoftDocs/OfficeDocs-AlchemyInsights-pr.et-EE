# <a name="unable-to-print"></a>Ei saa printida

Oleme teadlikud probleemi, kus mõned kasutajad ei saa printida, kasutades Safari või Edge brauserid. Kuigi me töötame Fix selle, palun kasutage teist brauserit. Vabandame ebamugavuste pärast.
