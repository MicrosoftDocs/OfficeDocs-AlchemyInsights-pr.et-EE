# <a name="unable-to-print"></a>Ei saa printida

Me teab probleemist, mille mõned kasutajad ei saa printida kui Safari või Edge brauseriga. Kui me tööd määrata seda, palun kasutage mõnda muud brauserit. Palume vabandust võimalike ebamugavuste pärast.
