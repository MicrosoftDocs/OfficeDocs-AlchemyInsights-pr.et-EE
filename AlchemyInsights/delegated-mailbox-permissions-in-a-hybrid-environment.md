# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Delegeeritud postkasti õigused hübriid keskkonnas

Sõltuvalt Exchange ' i versiooni olete installinud asutusesisesesse organisatsiooni, peate võib-olla võtta mõned täiendavad sammud delegeeritud postkasti õiguste seadistamiseks Asutusesisese organisatsiooni ja Exchange Online ' i Microsoft 365. Delegeeritud õiguste häälestamise kohta lisateabe saamiseks vaadake [konfigureerimine Exchange toetada delegeeritud postkasti õigused hübriid juurutamine](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
