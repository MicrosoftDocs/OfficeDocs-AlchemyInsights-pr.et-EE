# <a name="restore-items-deleted-from-sharepoint"></a>SharePointist kustutatud üksuste taastamine

Kui kustutate üksuse või saidi Sharepoint, see pole&rsquo;t kohe eemaldada. Kustutatud üksused lähevad prügikasti aja. Selle aja jooksul saate taastada kustutatud üksused nende algses asukohas. Lisateabe saamiseks külastage alltoodud linke.</p> <p><strong>Taastada kustutatud faile, kaustu ja saitide prügikasti.</strong></p> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/restore-deleted-items-from-the-site-collection-recycle-bin-5fa924ee-16d7-487b-9a0a-021b9062d14b?ui=en-US&amp;rs=en-US&amp;ad=US">SharePointi saidi prügikast üksuste taastamine</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://support.office.com/en-us/article/Restore-deleted-files-or-folders-in-OneDrive-949ada80-0026-4db3-a953-c99083e6a84f">Taastada kustutatud faile ja kaustu OneDrive'is</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/sharepoint/restore-deleted-site-collection">Taastada kustutatud saidikogumi (sealhulgas rühma-, side- ja mujal)</a></li> </ul> <ul> <li style="font-weight: 400;"><a href="https://docs.microsoft.com/en-us/onedrive/restore-deleted-onedrive">Taastada kustutatud OneDrive'i saidi</a></li> </ul> <p>Recycle bin hulgitoiminguid administraatoritel võib kaaluge <a href="https://docs.microsoft.com/en-us/powershell/sharepoint/sharepoint-pnp/sharepoint-pnp-cmdlets?view=sharepoint-ps">Sharepoint Online PNP.</a></p> <p><strong>Funktsiooni Taasta failid</strong></p> <p>Kui palju teie OneDrive'i või SharePointi faile kustutada, saada üle kirjutada, rikutud või nakatunud pahavara, saate taastada kogu OneDrive'i või SharePointi teegi eelmisele näidule&nbsp;failide taastamise funktsiooni abil.</p> <ul> <li><a href="https://support.office.com/en-us/article/restore-your-onedrive-fa231298-759d-41cf-bcd0-25ac53eb8a15">OneDrive teegi taastada</a></li> <li><a href="https://support.office.com/en-us/article/restore-a-document-library-317791c3-8bd0-4dfd-8254-3ca90883d39a?ui=en-US&amp;rs=en-US&amp;ad=US">Dokumendi taastamine</a></li> </ul> <p><strong>Märkus:</strong> SharePoint Online'i säilitab varukoopiaid sisu tegelik kustutamise üle 14 päeva. Kui sisu ei saa taastada kaudu prügikasti või faile taastada, administraator pöörduda Microsoft Support nõuda igal ajal 14-päevase aknas taastamise. &nbsp;Varukoopiatest taastamine lõpule viia ainult saidikogumite ja saitide, mitte jaoks kindlad failid, loenditele ja teekidele.</p>
