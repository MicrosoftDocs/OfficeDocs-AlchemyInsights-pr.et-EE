# <a name="windows-defender-actions-in-intune"></a>Windows Defenderi toimingud Intune

Intune ' i abil saate käivitada üksikutel seadmetel tellitavate skannimise ja viiruse signatuuri värskenduse Windows Defenderi jaoks.

Pärast seda, kui pult on edukalt käivitatud, kajastub tegevus Windows Defenderi sündmuselogisse.

Windowsi lõpp-punkti kaitsepoliitika võimaldab luua lisasätteid Windows Defenderi funktsioonide loomiseks Intune ' is ja rakendada seadmete kogumitele.

Lisateavet Windows Defenderi toimingute käivitamise kohta leiate teemast [tellitavate Microsoft Defenderi viirusetõrje skannimise konfigureerimine ja käitamine](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).
