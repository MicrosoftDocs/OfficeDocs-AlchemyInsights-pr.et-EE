# <a name="clear-the-browsing-data-stored-in-the-cloud-if-youve-turned-on-sync-or-personalization"></a>Pilves talletatud sirvimisandmete tühjendamine (kui olete sünkroonimise või isikupärastamise sisse lülitanud)

Pilvteenusesse salvestatud andmete vaatamiseks, haldamiseks või kustutamiseks kasutage privaatsuse armatuurlauda. Armatuurlaua abil kustutatud andmeid ei kustutata teie seadmest. Lisateavet leiate teemast [Andmete kuvamine privaatsuse armatuurlaual.](https://support.microsoft.com/windows/view-your-data-on-the-privacy-dashboard-03d3e27f-1981-5ff4-ba1c-d6b1031ae433)
