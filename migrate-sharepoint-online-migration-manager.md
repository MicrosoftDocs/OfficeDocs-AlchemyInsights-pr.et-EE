# <a name="migrating-to-sharepoint-online-via-migration-manager"></a>Migreerimine SharePoint Online ' i kaudu Migration Manager

Asub kaasaegne SharePointi administreerimiskeskus, migreerimise haldur juhendab teid oma klientide seadistamise ja oma ülesannete loomine. Saate määrata globaalsed või ülesandetaseme sätted, vaadata kogu ülesande edenemist ning alla laadida kokkuvõtlikud kokkuvõtte-ja ülesandetaseme aruanded.

* [Migreerimise halduri alustamine](https://docs.microsoft.com/sharepointmigration/mm-get-started)
* [Migreerimise haldur klientide seadistamine](https://docs.microsoft.com/sharepointmigration/mm-setup-clients)
* [Migreerimishalduri sätted](https://docs.microsoft.com/sharepointmigration/mm-settings)
